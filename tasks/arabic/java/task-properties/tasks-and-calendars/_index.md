# إضافة مهمة إلى المشروع وإدارة التقويمات باستخدام Aspose.Tasks للـ Java

## مقدمة
هل أنت مستعد **لإضافة مهمة إلى المشروع** والحفاظ على جدولك متناسقًا تمامًا؟ في هذا الدليل سنستعرض الخطوات الأساسية لإنشاء المهام، وربطها بتقويمات مخصصة، واستخدام Aspose.Tasks—وهي **مكتبة إدارة مشاريع جافا** رائدة في الصناعة. في النهاية ستعرف بالضبط كيفية **إنشاء تقويم مهمة بأسلوب جافا**، مما يمنحك تحكمًا دقيقًا في جداول المشروع.

## إجابات سريعة
- **ما هو الهدف الأساسي؟** إضافة مهمة إلى المشروع وربطها بتقويم مخصص.  
- **أي مكتبة يجب أن أستخدمها؟** Aspose.Tasks للـ Java – واجهة برمجة تطبيقات إدارة مشاريع كاملة المميزات.  
- **هل أحتاج إلى ترخيص؟** نسخة تجريبية مجانية متاحة؛ يلزم ترخيص تجاري للإنتاج.  
- **ما نسخة JDK المطلوبة؟** Java 8 أو أعلى.  
- **كم يستغرق التنفيذ؟** عادةً أقل من 15 دقيقة للسيناريوهات الأساسية.

## ما هو “إضافة مهمة إلى المشروع”؟
إضافة مهمة إلى مشروع يعني إنشاء عنصر عمل داخل كائن Project وتحديد خصائصه (المدة، تاريخ البدء، التقويم، إلخ). هذه العملية هي لبنة البناء لأي تطبيق يركز على الجدولة.

## لماذا نستخدم مكتبة إدارة مشاريع جافا؟
مكتبة مخصصة مثل Aspose.Tasks تتعامل مع تنسيق ملفات MS‑Project المعقد، وتوازن الموارد، وحسابات التقويم نيابةً عنك. إنها توفر عليك إعادة اختراع العجلة وتضمن التوافق مع الأدوات القياسية في الصناعة.

## المتطلبات المسبقة
قبل الغوص في الشرح، تأكد من توفر المتطلبات التالية:
- Java Development Kit (JDK): تأكد من تثبيت Java على نظامك.
- مكتبة Aspose.Tasks: قم بتحميل وإدراج مكتبة Aspose.Tasks في مشروعك. يمكنك العثور على المكتبة [هنا](https://releases.aspose.com/tasks/java/).

## استيراد الحزم
في مشروع Java الخاص بك، استورد الحزم اللازمة لـ Aspose.Tasks:

```java
import com.aspose.tasks.Calendar;
import com.aspose.tasks.ChildTasksCollector;
import com.aspose.tasks.Project;
import com.aspose.tasks.Task;
import com.aspose.tasks.TaskUtils;
import com.aspose.tasks.Tsk;
```

## الخطوة 1: إعداد مشروعك
ابدأ بإنشاء مشروع Java جديد وإضافة ملف Aspose.Tasks JAR إلى مسار البناء. سيمنحك ذلك الوصول إلى الواجهة البرمجية الكاملة.

## الخطوة 2: كيفية إضافة مهمة إلى المشروع
أنشئ كائن `Project` جديدًا وأضف مهمة على المستوى الجذري باسم **Task1**. يوضح ذلك عملية “إضافة مهمة إلى المشروع” الأساسية.

```java
Project project = new Project();
Task tsk = project.getRootTask().getChildren().add("Task1");
```

## الخطوة 3: كيفية إنشاء تقويم مهمة في جافا
أضف تقويمًا قياسيًا إلى مشروعك. التقويمات تحدد أيام العمل، العطلات، وغيرها من قواعد الجدولة.

```java
Calendar cal = project.getCalendars().add("TaskCal1");
```

## الخطوة 4: ربط المهمة بالتقويم
اربط المهمة التي أنشأتها مسبقًا بالتقويم الجديد بحيث يحترم جدولها وقت العمل المحدد في التقويم.

```java
tsk.set(Tsk.CALENDAR, cal);
```

*نصيحة:* كرّر هذه الخطوات لكل مهمة وتقويم إضافي تحتاجهما. يمكنك أيضًا تخصيص استثناءات التقويم (مثل أيام غير العمل) باستخدام واجهة `Calendar`.

## المشكلات الشائعة والحلول
- **المهمة لا تعكس تغييرات التقويم:** تأكد من استدعاء `project.updateStructure()` بعد تعديل التقويمات.  
- **NullPointerException عند استدعاء `set`:** تحقق من أن التقويم قد أُضيف بنجاح إلى المشروع قبل تعيينه.  
- **تواريخ غير صحيحة بعد الاستيراد/التصدير:** استخدم `project.save("output.mpp")` وأعد الفتح للتأكد من بقاء البيانات.
