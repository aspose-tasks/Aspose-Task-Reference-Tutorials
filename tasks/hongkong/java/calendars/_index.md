# 如何使用 Aspose.Tasks 建立日曆

## 介紹

如果您是想 **如何建立日曆** 物件以推動專案排程的 Java 開發者，您來對地方了。在此中心，我們彙集了一系列 Aspose.Tasks 教學，帶您逐步了解 MS Project 日曆管理的各個面向——從建立基本日曆、提取工作時間、定義工作日，到將日曆更新為 MPP 格式。無論您是構建全新排程引擎或是擴充既有解決方案，這些步驟指南都能協助您快速且可靠地完成任務。

## 快速解答
- **日曆處理的主要類別是什麼？** `com.aspose.tasks.Calendar`
- **我可以從現有日曆中提取工作時間嗎？** 是 – 使用 `WorkingTime` 集合。
- **是否可以自訂工作日？** 當然，可以為每一天設定 `WeekDay` 物件。
- **如何將日曆更新為 MPP 格式？** 呼叫 `project.save("file.mpp", SaveFileFormat.MPP)`。
- **生產環境需要授權嗎？** 非評估部署需要付費的 Aspose.Tasks 授權。

## 什麼是 **如何建立日曆** 在 Aspose.Tasks 中？
在 Aspose.Tasks 中建立日曆即是實例化一個 `Calendar` 物件，將其附加至 `Project`，並設定其工作天、假期與例外。此物件成為 MS Project 檔案中所有工作排程計算的核心。

## 為何在 Java 中使用 Aspose.Tasks 進行日曆管理？
- **完整的 MS Project 相容性** – 日曆可在儲存與讀取時保持完整性。
- **細緻的控制** – 您可以以程式方式調整工作時間、加入例外，並操作假期。
- **無 COM 相容** – 純 Java，適合跨平台環境。
- **效能優化的 API** – 能以最小記憶體佔用處理大型專案檔案。

## 常見問題

**問：如何以程式方式 **如何建立日曆** 物件於 Aspose.Tasks？**  
答：實例化 `com.aspose.tasks.Calendar`，透過 `project.getCalendars().add(calendar)` 將其加入 `Project`，然後設定其 `WeekDays`、`Exceptions` 與 `WorkingTimes`。

**問：從現有日曆中 **如何提取工作時間** 的最佳方法是什麼？**  
答：使用 `Calendar.getWorkingTimes()` 集合；每個 `WorkingTime` 條目包含該日的開始與結束時間。

**問：我可以 **如何定義工作日** 於自訂工作週嗎？**  
答：可以。從日曆取得 `WeekDay` 物件（`calendar.getWeekDays()`），並為每個需要自訂的日子設定 `DayWorking` 與 `WorkingTimes`。

**問：在變更日曆後，有沒有簡單的方法 **如何更新 mpp** 檔案？**  
答：變更日曆後，呼叫 `project.save("UpdatedProject.mpp", SaveFileFormat.MPP)` 即可將變更寫回 MPP 檔案。

**問：每個日曆操作都需要單獨的授權嗎？**  
答：不需要。單一的 Aspose.Tasks 授權即可涵蓋所有與日曆相關的 API，包括建立、提取、定義與 MPP 更新。
