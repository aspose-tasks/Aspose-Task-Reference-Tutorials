## 常見問題

**Q: 我可以使用 Aspose.Tasks 更新現有 MS Project 檔案的預算嗎？**  
A: 可以，API 允許您開啟檔案、修改任何指派的 Budget 欄位，並在不安裝 MS Project 的情況下儲存變更。

**Q: 指派的數量有上限嗎？**  
A: Aspose.Tasks 能有效處理數千筆指派，效能主要受可用記憶體限制。

**Q: 必須自行處理貨幣轉換嗎？**  
A: 函式庫僅處理數值型預算，貨幣換算需自行在程式中實作。

**Q: 如何讀取特定資源指派的預算？**  
A: 載入專案後，依 ID 或 Resource UID 取得 Assignment，然後呼叫 `assignment.getBudget()` 取得數值。

**Q: 需要哪個版本的 Aspose.Tasks 才能支援預算管理？**  
A: 所有近期發行版（2023‑2025）皆支援預算屬性，建議使用最新穩定版以獲得最佳相容性。
