# จัดการตัวเลขสกุลเงินของ ms project ด้วย Aspose.Tasks

## บทนำ
ในบทแนะนำที่ครอบคลุมนี้ คุณจะได้ค้นพบ **วิธีทำงานกับค่าสตางค์ของ ms project** โดยใช้ไลบรารี Aspose.Tasks สำหรับ Java ไม่ว่าคุณจะกำลังสร้างเครื่องมือรายงาน, ยูทิลิตี้การย้ายข้อมูล, หรือเพียงแค่ต้องการอ่านการตั้งค่าสกุลเงินจาก **ไฟล์ java project** คู่มือฉบับนี้จะพาคุณผ่านทุกขั้นตอน—from การโหลดไฟล์ *.mpp* ไปจนถึงการดึงตัวเลขสกุลเงิน. เมื่อเสร็จสิ้น คุณจะรู้สึกสบายใจในการจัดการข้อมูลสกุลเงินของ ms project ในแอปพลิเคชันของคุณเอง.

## คำตอบอย่างรวดเร็ว
- **ไลบรารีอะไรที่อ่านไฟล์ MS Project?** Aspose.Tasks for Java.  
- **ต้องใช้บรรทัดโค้ดกี่บรรทัดเพื่อรับตัวเลขสกุลเงิน?** เพียงสามบรรทัดสั้น ๆ หลังจากโหลดโปรเจกต์แล้ว.  
- **ต้องใช้ไลเซนส์สำหรับการพัฒนาหรือไม่?** รุ่นทดลองฟรีใช้ได้สำหรับการทดสอบ; จำเป็นต้องมีไลเซนส์เชิงพาณิชย์สำหรับการใช้งานจริง.  
- **เวอร์ชัน Java ที่รองรับคืออะไร?** Java 8 หรือสูงกว่า (JDK ใดก็ได้ที่รัน Aspose.Tasks).  
- **ฉันสามารถดึงคุณสมบัติอื่นของ Project ได้หรือไม่?** ได้ – Aspose.Tasks เปิดเผยชุดเต็มของฟิลด์ Project (เช่น วันที่เริ่มต้น, อัตราต้นทุน, เป็นต้น).

## ms project currency คืออะไร?
`ms project currency` หมายถึงความแม่นยำเชิงตัวเลข (จำนวนตำแหน่งทศนิยม) ที่ Microsoft Project ใช้เมื่อแสดงค่าการเงิน มันถูกเก็บไว้ในไฟล์ Project เป็นคุณสมบัติ **CURRENCY_DIGITS** และกำหนดว่าจำนวนเงินจะแสดงเป็นจำนวนเต็ม, หนึ่งตำแหน่งทศนิยม, สองตำแหน่งทศนิยม ฯลฯ

## ทำไมต้องใช้ Aspose.Tasks สำหรับจัดการ ms project currency?
- **ไม่ต้องติดตั้ง Microsoft Project** – ทำงานโดยตรงกับไฟล์ *.mpp* บนแพลตฟอร์มใดก็ได้ที่รองรับ Java.  
- **ความปลอดภัยของประเภทข้อมูลที่แข็งแรง** – API ส่งคืนค่าที่มีประเภทอย่างชัดเจน ลดข้อผิดพลาดการแปลงข้อมูล.  
- **ประสิทธิภาพที่ปรับแต่ง** – โหลดโปรเจกต์ขนาดใหญ่ได้อย่างรวดเร็วและดึงเฉพาะฟิลด์ที่ต้องการ.  
- **ข้ามแพลตฟอร์ม** – รันบน Windows, Linux หรือ macOS โดยไม่ต้องแก้ไข.

## ข้อกำหนดเบื้องต้น
ก่อนที่คุณจะเริ่ม, โปรดตรวจสอบว่าคุณมีสิ่งต่อไปนี้:

1. **สภาพแวดล้อมการพัฒนา Java** – JDK 8 หรือใหม่กว่า ติดตั้งและกำหนดค่าแล้ว.  
2. **Aspose.Tasks for Java** – ดาวน์โหลด JAR ล่าสุดจากเว็บไซต์อย่างเป็นทางการ: [Aspose.Tasks for Java](https://releases.aspose.com/tasks/java/).  
3. **ความรู้พื้นฐานของ Java** – คุณควรคุ้นเคยกับการสร้างโปรเจกต์ Java, การเพิ่มไลบรารีภายนอก, และการรันเมธอด `main`.  

## นำเข้าแพ็กเกจ
ก่อนอื่น, นำเข้าคลาสที่เราต้องการใช้.  
```java
import java.io.IOException;
import com.aspose.tasks.Prj;
import com.aspose.tasks.Project;
import com.aspose.tasks.SaveFileFormat;
```

## ขั้นตอนที่ 1: กำหนดไดเรกทอรีข้อมูล
ระบุโฟลเดอร์ที่มี **ไฟล์ java project** (`*.mpp`).  
```java
String dataDir = "Your Data Directory";
```
เปลี่ยน `"Your Data Directory"` ให้เป็นเส้นทางแบบ absolute หรือ relative ที่ไฟล์ `project.mpp` อยู่.

## ขั้นตอนที่ 2: โหลดไฟล์ MPP  
ตอนนี้เราจะดู **วิธีโหลดไฟล์ mpp** ด้วย Aspose.Tasks.  
```java
Project project = new Project(dataDir + "project.mpp");
```
ตรวจสอบให้แน่ใจว่าชื่อไฟล์ตรงกันอย่างแม่นยำ; หากไม่ตรง จะเกิด `IOException`.

## ขั้นตอนที่ 3: ดึงตัวเลขสกุลเงิน  
เมื่อโปรเจกต์ถูกโหลดแล้ว, การดึง **ตัวเลขสกุลเงินของ ms project** ทำได้ในบรรทัดเดียว:  
```java
System.out.println(project.get(Prj.CURRENCY_DIGITS));
```
เมธอดนี้คืนค่า `Integer` ที่แสดงจำนวนตำแหน่งทศนิยม (เช่น `2` สำหรับเซนต์). ค่าจะถูกพิมพ์ลงคอนโซล, แต่คุณก็สามารถเก็บไว้ในตัวแปรเพื่อการประมวลผลต่อได้.

## ปัญหาทั่วไปและเคล็ดลับ
- **ไฟล์ไม่พบ** – ตรวจสอบเส้นทาง `dataDir` อีกครั้งและให้แน่ใจว่าชื่อไฟล์ถูกต้อง รวมถึงนามสกุล `.mpp`.  
- **เวอร์ชันไฟล์ที่ไม่รองรับ** – Aspose.Tasks รองรับรูปแบบ Project 2000‑2024; ไฟล์เก่าหรือเสียหายอาจต้องแปลง.  
- **ไม่ได้ตั้งค่าไลเซนส์** – ในระหว่างการพัฒนารุ่นทดลองทำงานได้ แต่สำหรับการผลิตคุณต้องใช้ไลเซนส์ที่ถูกต้องเพื่อหลีกเลี่ยงลายน้ำการประเมินผล.

## คำถามที่พบบ่อย

**Q: Aspose.Tasks สามารถจัดการคุณลักษณะอื่นของ Project นอกจากตัวเลขสกุลเงินได้หรือไม่?**  
A: ได้, Aspose.Tasks มีฟังก์ชันหลากหลายเพื่อจัดการด้านต่าง ๆ ของไฟล์ Project, เช่น งาน, ทรัพยากร, และฟิลด์กำหนดเอง.

**Q: Aspose.Tasks เหมาะกับแอปพลิเคชันระดับองค์กรหรือไม่?**  
A: แน่นอน, Aspose.Tasks ถูกออกแบบมาเพื่อรองรับความต้องการของโครงการระดับองค์กร, ให้ประสิทธิภาพสูงและสามารถขยายได้.

**Q: Aspose.Tasks รองรับการพัฒนาข้ามแพลตฟอร์มหรือไม่?**  
A: ใช่, คุณสามารถใช้ Aspose.Tasks for Java บนแพลตฟอร์มใดก็ได้ที่รองรับ Java Runtime Environment (Windows, Linux, macOS).

**Q: ฉันสามารถทดลองใช้ Aspose.Tasks ก่อนซื้อได้หรือไม่?**  
A: ใช่, คุณสามารถดาวน์โหลดรุ่นทดลองฟรีจาก [ที่นี่](https://releases.aspose.com/).

**Q: จะหาการสนับสนุนสำหรับ Aspose.Tasks ได้จากที่ไหน?**  
A: คุณสามารถหาได้จาก [ฟอรัม Aspose.Tasks](https://forum.aspose.com/c/tasks/15).
