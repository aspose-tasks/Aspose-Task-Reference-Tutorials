## Vanliga frågor

**Q: Kan jag använda Aspose.Tasks för att uppdatera budgetar i en befintlig MS Project‑fil?**  
A: Ja, API‑et låter dig öppna, ändra Budget‑fältet för vilket uppdrag som helst och spara ändringarna utan att behöva ha MS Project installerat.

**Q: Finns det en gräns för hur många uppdrag jag kan hantera?**  
A: Aspose.Tasks hanterar tusentals uppdrag effektivt; prestandan beror främst på tillgängligt minne.

**Q: Måste jag hantera valutakonvertering manuellt?**  
A: Biblioteket arbetar med det numeriska budgetvärdet; du måste själv tillämpa eventuell valutakonvertering i din egen kod.

**Q: Hur läser jag budgeten för ett specifikt resursuppdrag?**  
A: Läs in projektet, lokalisera Assignment med dess ID eller Resource UID, och använd sedan `assignment.getBudget()` (Java‑metod) för att hämta värdet.

**Q: Vilken version av Aspose.Tasks krävs för budgethantering?**  
A: Alla senaste versioner (2023‑2025) stödjer budgetegenskaper; vi rekommenderar att du använder den senaste stabila versionen för bästa kompatibilitet.
