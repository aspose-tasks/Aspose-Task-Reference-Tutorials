# Обработка цифр валюты MS Project с помощью Aspose.Tasks

## Введение
В этом полном руководстве вы узнаете **как работать со значениями валюты MS Project** с использованием библиотеки Aspose.Tasks для Java. Независимо от того, создаёте ли вы инструмент отчётности, утилиту миграции или просто нужно прочитать настройки валюты из **java‑файла проекта**, это руководство проведёт вас через каждый шаг — от загрузки файла *.mpp* до извлечения цифр валюты. К концу вы будете уверенно обрабатывать данные валюты MS Project в своих приложениях.

## Быстрые ответы
- **Какая библиотека читает файлы MS Project?** Aspose.Tasks for Java.  
- **Сколько строк кода требуется для получения цифр валюты?** Всего три короткие строки после загрузки проекта.  
- **Нужна ли лицензия для разработки?** Бесплатная пробная версия подходит для тестирования; коммерческая лицензия требуется для продакшна.  
- **Какая версия Java поддерживается?** Java 8 или выше (любой JDK, поддерживающий Aspose.Tasks).  
- **Можно ли получить другие свойства проекта?** Да — Aspose.Tasks предоставляет полный набор полей проекта (например, дату начала, ставки затрат и т.д.).

## Что такое валюта MS Project?
`ms project currency` обозначает числовую точность (количество знаков после запятой), которую Microsoft Project использует при отображении денежных значений. Она хранится в файле проекта как свойство **CURRENCY_DIGITS** и определяет, будут ли суммы отображаться как целые числа, с одним, двумя и более знаками после запятой.

## Почему стоит использовать Aspose.Tasks для работы с валютой MS Project?
- **Не требуется установка Microsoft Project** — работает напрямую с файлами *.mpp* на любой платформе, поддерживающей Java.  
- **Строгая типобезопасность** — API возвращает типизированные значения, уменьшая ошибки парсинга.  
- **Оптимизирована по производительности** — быстро загружает большие проекты и извлекает только нужные поля.  
- **Кроссплатформенность** — работает на Windows, Linux и macOS без модификаций.

## Предварительные требования
Прежде чем начать, убедитесь, что у вас есть следующее:

1. **Среда разработки Java** — установленный и настроенный JDK 8 или новее.  
2. **Aspose.Tasks for Java** — скачайте последнюю JAR‑файл с официального сайта: [Aspose.Tasks for Java](https://releases.aspose.com/tasks/java/).  
3. **Базовые знания Java** — вы должны уметь создавать Java‑проект, подключать внешние библиотеки и запускать метод `main`.  

## Импорт пакетов
Сначала импортируем необходимые классы.  
```java
import java.io.IOException;
import com.aspose.tasks.Prj;
import com.aspose.tasks.Project;
import com.aspose.tasks.SaveFileFormat;
```

## Шаг 1: Определите каталог данных
Укажите папку, содержащую ваш **java‑файл проекта** (`*.mpp`).  
```java
String dataDir = "Your Data Directory";
```
Замените `"Your Data Directory"` на абсолютный или относительный путь, где находится `project.mpp`.

## Шаг 2: Загрузка файла MPP  
Теперь посмотрим, **как загрузить файлы mpp** с помощью Aspose.Tasks.  
```java
Project project = new Project(dataDir + "project.mpp");
```
Убедитесь, что имя файла указано точно; иначе будет выброшено `IOException`.

## Шаг 3: Получение цифр валюты  
После загрузки проекта извлечение **цифр валюты ms project** занимает одну строку:  
```java
System.out.println(project.get(Prj.CURRENCY_DIGITS));
```
Вызов возвращает `Integer`, представляющий количество знаков после запятой (например, `2` для центов). Значение выводится в консоль, но его также можно сохранить в переменной для дальнейшей обработки.

## Распространённые проблемы и советы
- **Файл не найден** — дважды проверьте путь `dataDir` и убедитесь, что имя файла указано правильно, включая расширение `.mpp`.  
- **Неподдерживаемая версия файла** — Aspose.Tasks поддерживает форматы Project 2000‑2024; более старые или повреждённые файлы могут потребовать конвертации.  
- **Лицензия не установлена** — в режиме разработки работает пробная версия, но для продакшна необходимо применить действующую лицензию, чтобы избавиться от водяных знаков оценки.
