## Часто задаваемые вопросы

**Q: Можно ли использовать Aspose.Tasks для обновления бюджетов в существующем файле MS Project?**  
A: Да, API позволяет открыть файл, изменить поле Budget любого назначения и сохранить изменения без необходимости установки MS Project.

**Q: Есть ли ограничение на количество назначений, которые можно управлять?**  
A: Aspose.Tasks эффективно обрабатывает тысячи назначений; производительность в основном зависит от доступной памяти.

**Q: Нужно ли вручную выполнять конвертацию валют?**  
A: Библиотека работает с числовым значением бюджета; любые конвертации валют необходимо выполнять в вашем коде.

**Q: Как прочитать бюджет конкретного назначения ресурса?**  
A: Загрузите проект, найдите Assignment по его ID или Resource UID, затем используйте `assignment.getBudget()` (метод Java) для получения значения.

**Q: Какая версия Aspose.Tasks требуется для управления бюджетом?**  
A: Все последние релизы (2023‑2025) поддерживают свойства бюджета; рекомендуется использовать последнюю стабильную версию для лучшей совместимости.
