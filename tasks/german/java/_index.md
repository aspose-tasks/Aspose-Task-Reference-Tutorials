- **Was bedeutet “create project calendar aspose”?** Es bezieht sich auf die Verwendung von Aspose.Tasks für Java zum Definieren, Ändern und Abrufen von Kalenderdaten in Microsoft‑Project‑Dateien.

## Häufig gestellte Fragen

**Q: Kann ich Aspose.Tasks für Java in einer kommerziellen Anwendung verwenden?**  
A: Ja, Sie können es kommerziell mit einer gültigen Aspose‑Lizenz verwenden. Eine kostenlose Testversion ist für die Evaluierung verfügbar.

**Q: Welche Java‑Versionen werden unterstützt?**  
A: Aspose.Tasks für Java unterstützt Java 8, 11 und neuere Versionen.

**Q: Wie füge ich programmgesteuert eine Kalenderausnahme hinzu?**  
A: Verwenden Sie die `Calendar`‑Klasse, um ein `Exception`‑Objekt zu erstellen, dessen Start‑/End‑Datum festzulegen und es zur Kalender‑Sammlung des Projekts hinzuzufügen.

**Q: Ist es möglich, Gantt‑Diagramm‑Balkenstile per Code anzupassen?**  
A: Absolut – Aspose.Tasks stellt das `GanttChartView`‑Objekt bereit, über das Sie Balkenfarben, Muster und weitere visuelle Attribute setzen können.

**Q: Wo finde ich die aktuelle API‑Dokumentation?**  
A: Die offizielle Dokumentation wird auf der Aspose‑Website im Abschnitt Aspose.Tasks für Java bereitgestellt.
