## Häufig gestellte Fragen

**Q: Kann ich Aspose.Tasks verwenden, um Budgets in einer bestehenden MS‑Project‑Datei zu aktualisieren?**  
A: Ja, die API ermöglicht es Ihnen, das Budget‑Feld einer beliebigen Zuweisung zu öffnen, zu ändern und die Änderungen zu speichern, ohne dass MS Project installiert sein muss.

**Q: Gibt es ein Limit für die Anzahl der Zuweisungen, die ich verwalten kann?**  
A: Aspose.Tasks verarbeitet tausende von Zuweisungen effizient; die Leistung hängt hauptsächlich vom verfügbaren Arbeitsspeicher ab.

**Q: Muss ich die Währungsumrechnung manuell durchführen?**  
A: Die Bibliothek arbeitet mit dem numerischen Budgetwert; Sie müssen etwaige Währungsumrechnungen in Ihrem eigenen Code vornehmen.

**Q: Wie lese ich das Budget einer bestimmten Ressourcenzuweisung?**  
A: Laden Sie das Projekt, finden Sie die Zuweisung anhand ihrer ID oder Resource‑UID und verwenden Sie dann `assignment.getBudget()` (Java‑Methode), um den Wert abzurufen.

**Q: Welche Version von Aspose.Tasks wird für die Budgetverwaltung benötigt?**  
A: Alle aktuellen Releases (2023‑2025) unterstützen Budget‑Eigenschaften; wir empfehlen die neueste stabile Version für beste Kompatibilität zu verwenden.
