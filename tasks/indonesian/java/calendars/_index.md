## Pertanyaan yang Sering Diajukan

**Q: Bagaimana cara saya secara programatis membuat objek **how to create calendar** di Aspose.Tasks?**  
A: Instansiasi `com.aspose.tasks.Calendar`, tambahkan ke `Project` melalui `project.getCalendars().add(calendar)`, lalu konfigurasikan `WeekDays`, `Exceptions`, dan `WorkingTimes`‑nya.

**Q: Apa cara terbaik untuk **how to extract hours** dari kalender yang ada?**  
A: Gunakan koleksi `Calendar.getWorkingTimes()`; setiap entri `WorkingTime` berisi waktu mulai dan selesai untuk hari tertentu.

**Q: Bisakah saya **how to define weekdays** untuk minggu kerja khusus?**  
A: Ya. Ambil objek `WeekDay` dari kalender (`calendar.getWeekDays()`) dan atur `DayWorking` serta `WorkingTimes` untuk setiap hari yang ingin Anda sesuaikan.

**Q: Apakah ada metode sederhana untuk **how to update mpp** file setelah mengubah kalender?**  
A: Setelah memodifikasi kalender, panggil `project.save("UpdatedProject.mpp", SaveFileFormat.MPP)` untuk menulis perubahan kembali ke file MPP.

**Q: Apakah saya memerlukan lisensi terpisah untuk setiap operasi kalender?**  
A: Tidak. Satu lisensi Aspose.Tasks mencakup semua API terkait kalender, termasuk pembuatan, ekstraksi, definisi, dan pembaruan MPP.
