## Pertanyaan yang Sering Diajukan

**Q: Bisakah saya menggunakan Aspose.Tasks untuk memperbarui anggaran dalam file MS Project yang sudah ada?**  
A: Ya, API memungkinkan Anda membuka, memodifikasi bidang Budget dari penugasan mana pun, dan menyimpan perubahan tanpa perlu menginstal MS Project.

**Q: Apakah ada batasan jumlah penugasan yang dapat saya kelola?**  
A: Aspose.Tasks menangani ribuan penugasan secara efisien; kinerja terutama bergantung pada memori yang tersedia.

**Q: Apakah saya harus menangani konversi mata uang secara manual?**  
A: Perpustakaan bekerja dengan nilai numerik anggaran; Anda perlu menerapkan konversi mata uang dalam kode Anda sendiri.

**Q: Bagaimana cara membaca anggaran dari penugasan sumber daya tertentu?**  
A: Muat proyek, temukan Assignment berdasarkan ID atau Resource UID, kemudian gunakan `assignment.getBudget()` (metode Java) untuk mengambil nilainya.

**Q: Versi Aspose.Tasks mana yang diperlukan untuk manajemen anggaran?**  
A: Semua rilis terbaru (2023‑2025) mendukung properti anggaran; kami menyarankan menggunakan versi stabil terbaru untuk kompatibilitas terbaik.
