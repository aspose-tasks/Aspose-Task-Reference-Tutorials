# Πώς να ορίσετε επανάληψη στο Aspose.Tasks (Μήνας, Εβδομάδα, Ημέρα)

## Εισαγωγή

Αν χρειάζεστε **πώς να ορίσετε επανάληψη** για εργασίες σε ένα έργο, το Aspose.Tasks for .NET σας παρέχει έναν καθαρό, προγραμματιστικό τρόπο για να προσθέσετε ορισμούς επαναλαμβανόμενων εργασιών που εκτελούνται ανά μήνα, εβδομάδα ή ημέρα. Σε αυτό το σεμινάριο θα περάσουμε από ένα πραγματικό παράδειγμα που δείχνει πώς να **προσθέσετε επαναλαμβανόμενες εργασίες**, **αυτοματοποιήσετε επαναλαμβανόμενες εργασίες**, και να τις διαχειριστείτε απευθείας από κώδικα C#. Στο τέλος, θα είστε έτοιμοι να ενσωματώσετε αυτή τη δυνατότητα σε οποιαδήποτε λύση χρονοπρογραμματισμού ή διαχείρισης έργων.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “recurrence” στο Aspose.Tasks;** Ορίζει ένα μοτίβο (καθημερινά, εβδομαδιαία, μηνιαία) που δημιουργεί αυτόματα στιγμιότυπα εργασιών σε μια χρονική περίοδο.  
- **Ποια κύρια μέθοδος δημιουργεί την επανάληψη;** `RecurringTaskParameters` σε συνδυασμό με ένα συγκεκριμένο `RecurrencePattern`.  
- **Χρειάζομαι άδεια για την εκτέλεση αυτού του κώδικα;** Η δοκιμαστική έκδοση λειτουργεί για αξιολόγηση· απαιτείται εμπορική άδεια για παραγωγή.  
- **Μπορώ να προγραμματίσω εβδομαδιαίες εργασίες αντί για μηνιαίες;** Ναι – αντικαταστήστε το `MonthlyRecurrencePattern` με `WeeklyRecurrencePattern`.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6 και μεταγενέστερες.

## Τι είναι η επανάληψη στο Aspose.Tasks;

Η επανάληψη είναι ένα σύνολο κανόνων που λέει στο Aspose.Tasks να δημιουργεί στιγμιότυπα εργασιών σε τακτά διαστήματα—καθημερινά, εβδομαδιαία ή μηνιαία—χωρίς να χρειάζεται να τις αντιγράψετε χειροκίνητα. Αυτή η δυνατότητα είναι ουσιώδης για έργα που περιλαμβάνουν ρουτινικές δραστηριότητες όπως συναντήσεις κατάστασης, επιθεωρήσεις ή εργασίες συντήρησης.

## Γιατί να χρησιμοποιήσετε τις δυνατότητες επανάληψης;

- **Εξοικονόμηση χρόνου:** Δεν χρειάζεται να αντιγράφετε‑επικολλείτε εργασίες χειροκίνητα.  
- **Μείωση σφαλμάτων:** Η βιβλιοθήκη εγγυάται συνεπείς ημερομηνίες και διάρκειες.  
- **Ευελιξία:** Συνδυάστε μοτίβα (π.χ., “πρώτη Κυριακή κάθε 2 μηνών”).  
- **Αυτοματοποίηση:** Ιδανικό για τη δημιουργία χρονοδιαγραμμάτων σε CI pipelines ή εργαλεία αναφοράς.

## Προαπαιτούμενα

1. **Βασική κατανόηση της C#** – θα γράψετε μερικές γραμμές κώδικα C#.  
2. **Εγκατεστημένο Aspose.Tasks for .NET** – κατεβάστε το από τη [σελίδα λήψης](https://releases.aspose.com/tasks/net/).  
3. **Ένα αρχείο έργου .mpp** – θα χρησιμοποιήσουμε το `Project1.mpp` ως αρχείο προέλευσης.

## Εισαγωγή ονοματοχώρων

Για να ξεκινήσετε, εισάγετε τα απαιτούμενα ονοματοχώρους του Aspose.Tasks:

```csharp
using Aspose.Tasks;
using System;

using Aspose.Tasks.Saving;
```

### Βήμα 1: Φόρτωση του αρχείου έργου

```csharp
// The path to th documents directory.
String DataDir = "Your Document Directory";
var project = new Project(DataDir + "Project1.mpp");
```

Δημιουργούμε μια παρουσία `Project` που δείχνει σε ένα υπάρχον αρχείο Microsoft Project.

### Βήμα 2: Ορισμός παραμέτρων επαναλαμβανόμενης εργασίας

```csharp
var parameters = new RecurringTaskParameters
{
    TaskName = "t1",
    Duration = project.GetDuration(1, TimeUnitType.Day),
    RecurrencePattern = new MonthlyRecurrencePattern
    {
        Repetition = new ByMonthWeekDayRepetition
        {
            Position = OrdinalNumber.First,
            WeekDay = DayOfWeek.Sunday,
            RepetitionInterval = 2
        },
        RecurrenceRange = new EndByRecurrenceRange
        {
            Start = new DateTime(2018, 7, 1, 8, 0, 0),
            Finish = new DateTime(2018, 9, 2, 17, 0, 0)
        }
    }
};
```

Εδώ **δημιουργούμε παραμέτρους επαναλαμβανόμενης εργασίας**:

- **TaskName** – το όνομα της παραγόμενης εργασίας.  
- **Duration** – τη διάρκεια κάθε εμφάνισης.  
- **RecurrencePattern** – ένα μηνιαίο μοτίβο που επαναλαμβάνεται κάθε 2 μήνες την πρώτη Κυριακή.  
- **RecurrenceRange** – τις ημερομηνίες έναρξης και λήξης που ορίζουν το χρονοδιάγραμμα.

### Βήμα 3: Προσθήκη της επαναλαμβανόμενης εργασίας στο έργο

```csharp
project.RootTask.Children.Add(parameters);
```

Αυτή η γραμμή **προσθέτει την επαναλαμβανόμενη εργασία** στη ρίζα της ιεραρχίας του έργου.

### Βήμα 4: Αποθήκευση του ενημερωμένου έργου

```csharp
project.Save(DataDir + "CanAddRecurringTask_Months_WeekDay_EndByRecurrenceRange_Test_out.mpp", SaveFileFormat.Mpp);
```

Το έργο αποθηκεύεται ως νέο αρχείο `.mpp` που τώρα περιέχει το αυτοματοποιημένο χρονοδιάγραμμα.

## Συχνά Προβλήματα και Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| **Η εργασία δεν εμφανίζεται** | Η περιοχή επανάληψης είναι εκτός των ημερομηνιών του έργου. | Επαληθεύστε ότι οι τιμές `Start` και `Finish` είναι εντός του ημερολογίου του έργου. |
| **Λάθος ημέρα της εβδομάδας** | Ασυμφωνία του enum `WeekDay`. | Χρησιμοποιήστε `DayOfWeek.Monday` … `DayOfWeek.Sunday` όπως απαιτείται. |
| **Αδυναμία άδειας** | Εκτέλεση χωρίς έγκυρη άδεια σε παραγωγή. | Εφαρμόστε προσωρινή ή πλήρη άδεια πριν από την αποθήκευση. |
