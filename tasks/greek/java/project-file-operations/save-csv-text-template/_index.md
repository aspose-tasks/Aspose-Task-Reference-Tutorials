# Αποθήκευση Έργου ως Πρότυπο, CSV και Κείμενο με Aspise.Tasks

## Εισαγωγή
Σε αυτό το σεμινάριο, θα έχετε **πώς να αποθηκεύσετε ένα έργο ως πρότυπο** και επίσης πώς να εξηγήσετε τα αρχεία Microsoft Project (MPP) σε μορφές CSV και απλού-κειμένου χρησιμοποιώντας τη βιβλιοθήκη Aspose.Tasks για Java. Πρέπει να δημιουργήσετε ένα επαναχρησιμοποιήσιμο πρότυπο έργου, να δημιουργήσετε αναφορές CSV για ανάλυση, είτε να παραγάγετε απλές εξηγήσεις κειμένου για ενσωμάτωση, αυτά τα βήματα που θα σας καθοδηγήσουν γρήγορα και αποτελεσματικά.

## Γρήγορες απαντήσεις
- **Μπορώ να εξάγω MPP σε CSV;** Ναι – χρησιμοποιήστε το «project.save(..., SaveFileFormat.CSV)».
- **Πώς να εξάγω κείμενο;** Αποθήκευση με το `SaveFileFormat.TEXT`.
- **Τι κάνει η «αποθήκευση έργου ως πρότυπο»;** Δημιουργεί ένα αρχείο `.mpt` που αφαιρεί τις πραγματικές και τις βασικές τιμές, έτοιμο για επαναχρησιμοποίηση.
- **Χρειάζομαι άδεια χρήσης;** Διατίθεται δοκιμαστική έκδοση. Απαιτείται εμπορική άδεια χρήσης για την παραγωγή.
- **Ποια έκδοση Java απαιτείται;** Υποστηρίζεται η Java8+.

## Τι είναι η «αποθήκευση έργου ως πρότυπο»;
Η αποθήκευση ενός έργου ως πρότυπο (`.mpt`) καταγράφει τη δομή, την ιεραρχία εργασιών και τις αναθέσεις πόρων, ενώ αφαιρεί τις πραγματικές αρχές έναρξης/λήξης και τα δεδομένα βάσης. Αυτό κάνει το τυπικό πρότυπο για επαναχρησιμοποίηση ενός σχεδίου έργου σε πολλά νέα έργα.

## Γιατί να χρησιμοποιήσετε το Aspose.Tasks για Java;
Το Aspose.Tasks σας επιτρέπει να χειρίζεστε αρχεία του Microsoft Project χωρίς να εγκαταστήσετε το ίδιο το Microsoft Project. Υποστηρίζει **πώς να εξηγήσω MPP**, **πώς να εξηγήσω κείμενο**, και **μετατροπή MPP σε CSV**, όλα από καθαρό κώδικα Java, κάτι που είναι τέλειο για αυτοματισμούς σε διακομιστές, CI pipelines ή επιτραπέζιες εφαρμογές.

## Προαπαιτούμενα
Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε:

1. Εγκατεστημένο Java Development Kit (JDK) 8 ή νεότερη έκδοση.
2. Η βιβλιοθήκη Aspose.Tasks for Java προστέθηκε στο έργο σας. Κατεβάστε το από [εδώ](https://releases.aspose.com/tasks/java/).
3. Βασική εξοικείωση με τη σύνταξη Java και τη ρύθμιση έργων Maven/Gradle.

## Εισαγωγή πακέτων
Πρώτα, εισαγάγετε τις απαιτούμενες κλάσεις στο αρχείο πηγαίου κώδικα Java:

```java
import java.io.IOException;
import com.aspose.tasks.*;
```

## Αποθήκευση Έργου ως CSV (Εξαγωγή MPP σε CSV)
Η εξαγωγή ενός αρχείου MPP σε CSV είναι χρήσιμη για την ανάλυση δεδομένων στο Excel ή σε εργαλεία BI.

### Βήμα 1: Φόρτωση του Έργου
```java
String projectName = "YourProject.mpp";
Project project = new Project(projectName);
```

### Βήμα 2: Αποθήκευση ως CSV
```java
String csvFileName = "output.csv";
project.save(csvFileName, com.aspose.tasks.SaveFileFormat.CSV);
```

## Αποθήκευση Έργου ως Κείμενο (Πώς να Εξάγετε Κείμενο)
Εάν χρειάζεστε μια αναπαράσταση εργασιών, πόρων ή αναθέσεων σε απλό κείμενο, αποθηκεύστε το έργο ως αρχείο κειμένου.

### Βήμα 1: Φόρτωση του Έργου
```java
String projectName = "YourProject.mpp";
Project project = new Project(projectName);
```

### Βήμα 2: Αποθήκευση ως Κείμενο
```java
String textFileName = "output.txt";
project.save(textFileName, com.aspose.tasks.SaveFileFormat.TEXT);
```

## Αποθήκευση Έργου ως Προτύπου (Δημιουργία Προτύπου Έργου Java)
Η δημιουργία ενός επαναχρησιμοποιήσιμου προτύπου καταργεί τις πραγματικές ημερομηνίες και τις γραμμές βάσης, αφήνοντας έναν καθαρό σκελετό για νέα έργα.

### Βήμα 1: Φόρτωση του Έργου
```java
String projectName = "YourProject.mpp";
Project project = new Project(projectName);
```

### Βήμα 2: Ορισμός Επιλογών Προτύπου
```java
SaveTemplateOptions options = new SaveTemplateOptions();
options.setRemoveActualValues(true);
options.setRemoveBaselineValues(true);
```

### Βήμα 3: Αποθήκευση ως Προτύπου
```java
String templateName = "output.mpt";
project.saveAsTemplate(templateName, options);
```

## Συνήθη Προβλήματα & Συμβουλές
- **Δεν Βρέθηκε Το Αρχείο:** Βεβαιωθείτε ότι η διαδρομή προς το `YourProject.mpp` είναι σωστή ή χρησιμοποιήστε μια απόλυτη διαδρομή.
- **Εξαιρέσεις Άδειας Χρήσης:** Χωρίς έγκυρη άδεια χρήσης, η βιβλιοθήκη εκτελείται σε λειτουργία αξιολόγησης και ενδέχεται να προσθέσει υδατογραφήματα. Εφαρμόστε την άδειά σας νωρίς στον κώδικα (`License license = new License(); license.setLicense("Aspose.Tasks.Java.lic");`).
- **Μεγάλα Έργα:** Για πολύ μεγάλα αρχεία MPP, εξετάστε το ενδεχόμενο αύξησης του μεγέθους του σωρού JVM (`-Xmx2g`) για να αποφύγετε το `OutOfMemoryError`.
