# Πώς να λάβετε τα πεδία πίνακα και να διαβάσετε τα δεδομένα πίνακα στο Aspose.Tasks

## Εισαγωγή
Σε αυτό το tutorial, θα ανακαλύψετε **πώς να λάβετε τα πεδία πίνακα** από ένα αρχείο Microsoft Project και να διαβάσετε τα δεδομένα πίνακα χρησιμοποιώντας το Aspose.Tasks for Java. Είτε δημιουργείτε εργαλεία αναφοράς, μεταφέρετε δεδομένα ή αυτοματοποιείτε αναλύσεις έργων, η εξαγωγή πληροφοριών πίνακα προγραμματιστικά εξοικονομεί ώρες χειροκίνητης εργασίας. Θα περάσουμε από όλη τη διαδικασία — από τη ρύθμιση του περιβάλλοντος μέχρι την εκτύπωση των λεπτομερειών κάθε πεδίου — ώστε να ενσωματώσετε αυτή τη δυνατότητα στις δικές σας εφαρμογές αμέσως.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “get table fields”;** Αναφέρεται στην ανάκτηση του ορισμού (πλάτος, τίτλος, στοίχιση κ.λπ.) κάθε στήλης που εμφανίζεται σε έναν πίνακα προβολής του Project.  
- **Ποια βιβλιοθήκη χρειάζεται;** Aspose.Tasks for Java.  
- **Χρειάζομαι άδεια για ανάπτυξη;** Μια δωρεάν δοκιμαστική έκδοση λειτουργεί για αξιολόγηση· απαιτείται εμπορική άδεια για παραγωγική χρήση.  
- **Μπορώ να διαβάσω πίνακες από οποιαδήποτε έκδοση του Project;** Ναι, το Aspose.Tasks υποστηρίζει μορφές Project 2003‑2016 και νεότερες.  
- **Απαιτείται επιπλέον ρύθμιση;** Απλώς JDK 8+ και το Aspose.Tasks JAR στο classpath σας.

## Προαπαιτούμενα
Πριν προχωρήσουμε, βεβαιωθείτε ότι έχετε τα εξής:

1. **Java Development Kit (JDK)** – Εγκατεστημένο JDK 8 ή νεότερο. Μπορείτε να το κατεβάσετε από την ιστοσελίδα της Oracle.  
2. **Aspose.Tasks for Java JAR** – Κατεβάστε τη νεότερη βιβλιοθήκη από το [download link](https://releases.aspose.com/tasks/java/) και προσθέστε την στο build path του έργου σας.  

## Εισαγωγή Πακέτων
Εισάγετε τις απαραίτητες κλάσεις του Aspose.Tasks:

```java
import com.aspose.tasks.Project;
import com.aspose.tasks.Table;
import com.aspose.tasks.TableField;
```

## Βήμα 1: Ρύθμιση του Καταλόγου Δεδομένων
Ορίστε το φάκελο που περιέχει το αρχείο *.mpp* σας:

```java
String dataDir = "Your Data Directory";
```

Αντικαταστήστε το `"Your Data Directory"` με την απόλυτη διαδρομή στο μηχάνημά σας (π.χ., `C:/Projects/Data/`).

## Βήμα 2: Φόρτωση του Αρχείου Project
Δημιουργήστε ένα αντικείμενο `Project` δείχνοντας στο αρχείο Project που θέλετε να εξετάσετε:

```java
Project project = new Project(dataDir + "Project2003.mpp");
```

Αν το αρχείο σας έχει διαφορετικό όνομα ή επέκταση, προσαρμόστε το string ανάλογα.

## Βήμα 3: Ανάκτηση πληροφοριών πίνακα
Τώρα θα **λάβουμε τα πεδία πίνακα** και θα εμφανίσουμε τις ιδιότητες κάθε πεδίου:

```java
Table t1 = project.getTables().toList().get(0);
System.out.println("Table Fields Count: " + t1.getTableFields().size());
System.out.println();
for (TableField f : t1.getTableFields()) {
    System.out.println("Field width: " + f.getWidth());
    System.out.println("Field Title: " + f.getTitle());
    System.out.println("Field Title Alignment: " + f.getAlignTitle());
    System.out.println("Field Align Data: " + f.getAlignData());
    System.out.println();
}
```

Το απόσπασμα κώδικα εκτυπώνει το πλάτος, τον τίτλο και τη στοίχιση για κάθε στήλη στον προεπιλεγμένο πίνακα, παρέχοντάς σας πλήρη εικόνα των **πεδίων πίνακα** που ορίζονται στο έργο.

## Γιατί να ανακτήσετε πληροφορίες πίνακα;
- **Automation** – Δημιουργία προσαρμοσμένων αναφορών χωρίς χειροκίνητη αντιγραφή‑επικόλληση.  
- **Migration** – Μεταφορά δεδομένων από παλαιά αρχεία Project σε σύγχρονες βάσεις δεδομένων.  
- **Validation** – Διασφάλιση ότι τα πρότυπα έργου συμμορφώνονται με τα οργανωτικά πρότυπα.  

## Συνηθισμένα Πιθανά Προβλήματα & Συμβουλές
- **Null tables** – Εάν ένα έργο δεν έχει πίνακες, το `project.getTables()` μπορεί να είναι κενό. Πάντα ελέγχετε το μέγεθος της λίστας πριν προσπελάσετε το δείκτη `0`.  
- **Encoding issues** – Οι μη‑ASCII χαρακτήρες στους τίτλους εμφανίζονται σωστά όταν χρησιμοποιείτε την πιο πρόσφατη έκδοση του Aspose.Tasks.  
- **Performance** – Η φόρτωση πολύ μεγάλων αρχείων *.mpp* μπορεί να απαιτεί πολλή μνήμη· σκεφτείτε τη χρήση streaming API για τεράστιες συλλογές δεδομένων.

## Επιπλέον Συχνές Ερωτήσεις

**Ε: Πώς να διαβάσω δεδομένα πίνακα σε περιβάλλον πολλαπλών έργων;**  
A: Φορτώστε κάθε έργο ξεχωριστά με `new Project(path)` και επαναλάβετε τον βρόχο εξαγωγής πεδίων πίνακα για κάθε παρουσία.

**Ε: Μπορώ να εξάγω τα ανακτημένα πεδία πίνακα σε CSV;**  
A: Ναι, μετά την εκτύπωση των λεπτομερειών των πεδίων μπορείτε να τα γράψετε σε ένα `FileWriter` ή να χρησιμοποιήσετε βιβλιοθήκη CSV όπως το OpenCSV.

**Ε: Το Aspose.Tasks διαχειρίζεται προσαρμοσμένους πίνακες που δημιουργούν οι χρήστες;**  
A: Απόλυτα. Η συλλογή `project.getTables()` περιλαμβάνει τόσο τους προεπιλεγμένους όσο και τους από τους χρήστες ορισμένους πίνακες, ώστε μπορείτε να τους διατρέξετε όπως χρειάζεται.

**Ε: Τι γίνεται αν το αρχείο Project είναι προστατευμένο με κωδικό πρόσβασης;**  
A: Χρησιμοποιήστε τον υπερφορτωμένο κατασκευαστή `Project` που δέχεται αντικείμενο `LoadOptions` όπου μπορείτε να ορίσετε τον κωδικό πρόσβασης.

**Ε: Υπάρχει τρόπος να φιλτράρω μόνο τις ορατές στήλες;**  
A: Ελέγξτε τη μέθοδο `getVisible()` του κάθε `TableField` (διαθέσιμη σε νεότερες εκδόσεις) για να προσδιορίσετε αν η στήλη εμφανίζεται στη διεπαφή χρήστη.
