# Αποθήκευση Έργου ως PDF – Απόδοση Φύλλου Εργασιών στο Aspose.Tasks

## Εισαγωγή
Σε αυτόν τον ολοκληρωμένο οδηγό θα ανακαλύψετε πώς να **αποθηκεύσετε το έργο ως PDF** και να δημιουργήσετε ένα φύλλο εργασιών με το Aspose.Tasks για Java. Είτε χρειάζεστε να εξάγετε ένα αρχείο MPP σε PDF για ανασκοπήσεις ενδιαφερόμενων, είτε απλώς να αποδώσετε ένα καθαρό φύλλο εργασιών για αναφορές, αυτό το tutorial σας καθοδηγεί βήμα προς βήμα, εξηγεί γιατί κάθε ρύθμιση είναι σημαντική και εμφανίζει τον ακριβή κώδικα που πρέπει να εκτελέσετε.

## Γρήγορες Απαντήσεις
- **Τι κάνει η “αποθήκευση έργου ως pdf”;** Μετατρέπει ένα αρχείο Microsoft Project (MPP) σε έγγραφο PDF, προαιρετικά μορφοποιώντας το ως φύλλο εργασιών.  
- **Ποια μορφή χρησιμοποιείται για PDF φύλλου εργασιών;** Η επιλογή `PresentationFormat.TaskSheet` στο `PdfSaveOptions`.  
- **Χρειάζομαι άδεια για την εκτέλεση του κώδικα;** Απαιτείται προσωρινή ή πλήρης άδεια για παραγωγή· μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση.  
- **Ποια είναι τα κύρια προαπαιτούμενα;** Java JDK, βιβλιοθήκη Aspose.Tasks για Java και ένα έγκυρο αρχείο έργου (π.χ., *.mpp).  
- **Μπορώ να προσαρμόσω τη διάταξη του PDF;** Ναι, το Aspose.Tasks προσφέρει πολλές ιδιότητες `PdfSaveOptions` για στυλ και ρύθμιση σελίδας.

## Τι είναι η “αποθήκευση έργου ως pdf”;
Η αποθήκευση ενός έργου ως PDF μετατρέπει τα εσωτερικά δεδομένα του έργου σε ένα φορητό, μόνο για ανάγνωση έγγραφο. Αυτό είναι ιδανικό για κοινή χρήση χρονοδιαγραμμάτων, φύλλων εργασιών ή αναφορών κατάστασης με ενδιαφερόμενους που δεν έχουν εγκατεστημένο το Microsoft Project.

## Γιατί να αποδώσετε ένα φύλλο εργασιών;
Ένα φύλλο εργασιών παρουσιάζει κάθε εργασία σε μορφή πίνακα, εμφανίζοντας τα ID, τα ονόματα, τις ημερομηνίες έναρξης/λήξης και άλλα πεδία. Η απόδοσή του σε PDF εξασφαλίζει συνεπή μορφοποίηση σε όλες τις πλατφόρμες και καθιστά εύκολη την εκτύπωση ή την ενσωμάτωση σε άλλα έγγραφα.

## Προαπαιτούμενα
Before you start, make sure you have:

1. **Java Development Kit (JDK)** – η τελευταία έκδοση εγκατεστημένη και τα `java`/`javac` διαθέσιμα στο PATH σας.  
2. **Aspose.Tasks for Java** – κατεβάστε τη βιβλιοθήκη από την επίσημη ιστοσελίδα [εδώ](https://releases.aspose.com/tasks/java/).  
3. Ένα έγκυρο **αρχείο έργου** (`*.mpp`) που θέλετε να μετατρέψετε, π.χ., `NewProductDev.mpp`.  

## Εισαγωγή Πακέτων
Αρχικά, εισάγετε τις απαιτούμενες κλάσεις του Aspose.Tasks. Αυτές οι εισαγωγές σας δίνουν πρόσβαση στις επιλογές αποθήκευσης PDF, στις μορφές παρουσίασης και στο βασικό αντικείμενο `Project`.

```java
import com.aspose.tasks.PdfSaveOptions;
import com.aspose.tasks.PresentationFormat;
import com.aspose.tasks.Project;
import com.aspose.tasks.SaveOptions;
```

## Οδηγός Βήμα‑βήμα

### Βήμα 1: Ορίστε τον Κατάλογο Εγγράφου
Ορίστε το φάκελο όπου βρίσκεται το αρχείο MPP προέλευσης και όπου θα γραφτεί το παραγόμενο PDF. Η χρήση απόλυτης ή σχετικής διαδρομής λειτουργεί· απλώς βεβαιωθείτε ότι ο φάκελος υπάρχει.

```java
String dataDir = "Your Document Directory";
```

### Βήμα 2: Φορτώστε το Αρχείο Έργου
Δημιουργήστε μια παρουσία `Project` φορτώνοντας το υπάρχον αρχείο MPP. Αυτό το αντικείμενο αντιπροσωπεύει ολόκληρη τη δομή του έργου στη μνήμη.

```java
Project project = new Project(dataDir + "NewProductDev.mpp");
```

### Βήμα 3: Διαμορφώστε τις Επιλογές Αποθήκευσης – Εξαγωγή MPP σε PDF
Δημιουργήστε μια παρουσία `PdfSaveOptions` και ενημερώστε το Aspose.Tasks ότι θέλετε το PDF να αποδοθεί ως **φύλλο εργασιών**. Αυτό είναι το βασικό βήμα για τη χρήση *εξαγωγής έργου σε pdf*.

```java
SaveOptions options = new PdfSaveOptions();
options.setPresentationFormat(PresentationFormat.TaskSheet);
```

### Βήμα 4: Αποθηκεύστε το Έργο ως Φύλλο Εργασιών – Αποθήκευση Έργου ως PDF
Κληθείτε τη μέθοδο `save` στο αντικείμενο `Project`, περνώντας το όνομα του αρχείου προορισμού και τις διαμορφωμένες επιλογές. Αυτή η ενέργεια **μετατρέπει το αρχείο MPP σε PDF** μορφοποιημένο ως φύλλο εργασιών.

```java
project.save(dataDir + "taskSheet.pdf", options);
```

### Βήμα 5: Ελέγξτε το Αποτέλεσμα
Πλοηγηθείτε στον κατάλογο που καθορίσατε (`dataDir`). Θα πρέπει να δείτε το `taskSheet.pdf` που περιέχει μια καλοσχεδιασμένη λίστα όλων των εργασιών. Ανοίξτε το με οποιονδήποτε προβολέα PDF για να επαληθεύσετε ότι η μετατροπή ολοκληρώθηκε επιτυχώς.

## Συχνά Προβλήματα και Λύσεις
- **FileNotFoundException** – Βεβαιωθείτε ότι το `dataDir` τελειώνει με διαχωριστικό διαδρομής (`/` ή `\\`) και ότι το όνομα του αρχείου MPP είναι σωστό.  
- **LicenseException** – Εάν εμφανίζονται σφάλματα άδειας, εφαρμόστε μια προσωρινή ή αγορασμένη άδεια πριν καλέσετε οποιοδήποτε API του Aspose.Tasks.  
- **Blank PDF** – Επαληθεύστε ότι το έργο περιέχει πραγματικά εργασίες· ένα κενό έργο θα παραγάγει κενό PDF.
