# Πώς να αποθηκεύσετε XML & να διαχειριστείτε το οικονομικό έτος στο Aspose.Tasks

## Εισαγωγή
Αν χρειάζεστε **how to save xml** αρχεία που δημιουργούνται από δεδομένα του Microsoft Project, το Aspose.Tasks for Java σας παρέχει έναν καθαρό, χωρίς άδεια τρόπο για να το κάνετε. Σε αυτό το σεμινάριο θα περάσουμε από τη φόρτωση ενός αρχείου MPP, την εμφάνιση πληροφοριών του οικονομικού έτους, τον ορισμό ιδιοτήτων του οικονομικού έτους και τελικά την **how to save xml** έξοδο. Θα δείτε επίσης πώς να **how to set fiscal** λεπτομέρειες και **how to convert mpp** αρχεία χωρίς ποτέ να ανοίξετε το Microsoft Project.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “manage fiscal year” στο Aspose.Tasks;** Σας επιτρέπει να ορίσετε το μήνα έναρξης του οικονομικού έτους και να ενεργοποιήσετε την αρίθμηση του οικονομικού έτους για ένα έργο.  
- **Πώς να ορίσετε το μήνα έναρξης του οικονομικού έτους;** Use the `Prj.FY_START_DATE` property with a `Month` enum value (e.g., `Month.JULY`).  
- **Μπορώ να φορτώσω ένα αρχείο MPP;** Ναι, απλώς δημιουργήστε μια παρουσία `Project` με τη διαδρομή προς το αρχείο *.mpp*.  
- **Πώς να μετατρέψετε MPP σε XML;** Καλείτε `project.save(..., SaveFileFormat.Xml)` μετά τον ορισμό των επιθυμητών ιδιοτήτων.  
- **Χρειάζομαι άδεια;** Διατίθεται δωρεάν δοκιμή· απαιτείται εμπορική άδεια για παραγωγική χρήση.

## Πώς να φορτώσετε ένα αρχείο MPP και να εμφανίσετε πληροφορίες του οικονομικού έτους
Παρακάτω θα αναλύσουμε τη διαδικασία σε σαφή, αριθμημένα βήματα.

### Βήμα 1: Φόρτωση Αρχείου Έργου
```java
// The path to the documents directory.
String dataDir = "Your Data Directory";
Project project = new Project(dataDir + "project.mpp");
```
Εδώ **load an MPP file** (`project.mpp`) από τον καθορισμένο φάκελο. Αυτό είναι το πρώτο βήμα σε οποιαδήποτε επεξεργασία σχετική με το οικονομικό έτος.

### Βήμα 2: Εμφάνιση Ιδιοτήτων Οικονομικού Έτους
```java
//Display fiscal year properties
System.out.println("Fiscal Year Start Date : " + project.get(Prj.FY_START_DATE));
System.out.println("Fiscal Year Numbering : " + project.get(Prj.FISCAL_YEAR_START));
```
Οι ιδιότητες `Prj.FY_START_DATE` και `Prj.FISCAL_YEAR_START` σας επιτρέπουν να **display fiscal year** λεπτομέρειες, απαντώντας στην ερώτηση “what is the current fiscal configuration?”.

### Βήμα 3: Ορισμός Έναρξης Οικονομικού Έτους και Ενεργοποίηση Αρίθμησης
```java
//Create a project instance
Project prj = new Project();
//Set fiscal year properties
prj.set(Prj.FY_START_DATE, Month.JULY);
prj.set(Prj.FISCAL_YEAR_START, new NullableBool(true));
//Save the project as XML project file
prj.save(dataDir + "savedProject.xml", SaveFileFormat.Xml);
```
Σε αυτό το βήμα κάνουμε **how to set fiscal** ρυθμίσεις:
- `Month.JULY` ορίζει το μήνα έναρξης του οικονομικού έτους.  
- `NullableBool(true)` ενεργοποιεί την αρίθμηση του οικονομικού έτους.  
- Τέλος, το έργο **converted MPP to XML** χρησιμοποιώντας `SaveFileFormat.Xml`.

### Βήμα 4: Επιβεβαίωση Λειτουργίας
```java
//Display result of conversion.
System.out.println("Process completed Successfully");
```
Ένα απλό μήνυμα κονσόλας επιβεβαιώνει ότι το οικονομικό έτος έχει ρυθμιστεί και το αρχείο αποθηκεύτηκε.
