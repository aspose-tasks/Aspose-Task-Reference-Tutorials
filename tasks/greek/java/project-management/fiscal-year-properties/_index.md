# Διαχείριση Ιδιοτήτων Οικονομικού Έτους στο Aspose.Tasks

## Εισαγωγή
Το Aspose.Tasks είναι μια ισχυρή βιβλιοθήκη Java που επιτρέπει στους προγραμματιστές να **διαχειρίζονται ρυθμίσεις οικονομικού έτους**, να φορτώνουν αρχεία MPP και να μετατρέπουν δεδομένα έργου σε XML με λίγες μόνο γραμμές κώδικα. Σε αυτό το tutorial θα δείτε ακριβώς πώς να ορίσετε τις ιδιότητες του οικονομικού έτους, να εμφανίσετε πληροφορίες οικονομικού έτους και να αποθηκεύσετε το αποτέλεσμα — όλα ενώ διατηρείτε τον κώδικά σας καθαρό και εύκολο στη συντήρηση.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “διαχείριση οικονομικού έτους” στο Aspose.Tasks;** Σας επιτρέπει να ορίσετε το μήνα έναρξης του οικονομικού έτους και να ενεργοποιήσετε την αρίθμηση του οικονομικού έτους για ένα έργο.  
- **Πώς να ορίσω το μήνα έναρξης του οικονομικού έτους;** Χρησιμοποιήστε την ιδιότητα `Prj.FY_START_DATE` με μια τιμή του enum `Month` (π.χ., `Month.JULY`).  
- **Μπορώ να φορτώσω ένα αρχείο MPP;** Ναι, απλώς δημιουργήστε ένα αντικείμενο `Project` με τη διαδρομή προς το αρχείο *.mpp*.  
- **Πώς να μετατρέψω MPP σε XML;** Καλέστε `project.save(..., SaveFileFormat.Xml)` αφού ορίσετε τις επιθυμητές ιδιότητες.  
- **Χρειάζομαι άδεια;** Διατίθεται δωρεάν δοκιμαστική έκδοση· απαιτείται εμπορική άδεια για παραγωγική χρήση.

## Τι σημαίνει “διαχείριση οικονομικού έτους” σε αρχεία έργου;
Η διαχείριση του οικονομικού έτους σημαίνει τη διαμόρφωση του ημερολογίου που ακολουθεί το έργο σας για οικονομική αναφορά. Αυτό περιλαμβάνει τον ορισμό του μήνα έναρξης του οικονομικού έτους και, προαιρετικά, την ενεργοποίηση της αρίθμησης του οικονομικού έτους, η οποία επηρεάζει τον τρόπο υπολογισμού και εμφάνισης των ημερομηνιών σε αναφορές.

## Γιατί να χρησιμοποιήσετε το Aspose.Tasks για τη διαχείριση του οικονομικού έτους;
- **Δεν απαιτείται Microsoft Project** – εργάζεστε απευθείας με αρχεία έργου σε Java.  
- **Πλήρης έλεγχος** – ορίστε την έναρξη του οικονομικού έτους, ενεργοποιήστε την αρίθμηση και μετατρέψτε μορφές προγραμματιστικά.  
- **Ισχυρό API** – αξιόπιστη διαχείριση μεγάλων αρχείων MPP και απρόσκοπτη εξαγωγή σε XML.

## Προαπαιτούμενα

1. Java Development Kit (JDK) εγκατεστημένο στο σύστημά σας.  
2. Aspose.Tasks for Java JAR – κατεβάστε το από [εδώ](https://releases.aspose.com/tasks/java/) και προσθέστε το στο classpath του έργου σας.

## Εισαγωγή Πακέτων
Για να ξεκινήσετε, εισάγετε τις απαραίτητες κλάσεις στο αρχείο πηγαίου κώδικα Java:
```java
import com.aspose.tasks.*;
```

## Πώς να φορτώσετε ένα αρχείο MPP και να εμφανίσετε πληροφορίες οικονομικού έτους
Παρακάτω διασπάμε τη διαδικασία σε σαφή, αριθμημένα βήματα.

### Βήμα 1: Φόρτωση Αρχείου Έργου
```java
// The path to the documents directory.
String dataDir = "Your Data Directory";
Project project = new Project(dataDir + "project.mpp");
```
Εδώ **φορτώνουμε ένα αρχείο MPP** (`project.mpp`) από τον καθορισμένο φάκελο. Αυτό είναι το πρώτο βήμα σε οποιαδήποτε επεξεργασία σχετική με το οικονομικό έτος.

### Βήμα 2: Εμφάνιση Ιδιοτήτων Οικονομικού Έτους
```java
//Display fiscal year properties
System.out.println("Fiscal Year Start Date : " + project.get(Prj.FY_START_DATE));
System.out.println("Fiscal Year Numbering : " + project.get(Prj.FISCAL_YEAR_START));
```
Οι ιδιότητες `Prj.FY_START_DATE` και `Prj.FISCAL_YEAR_START` σας επιτρέπουν να **εμφανίσετε λεπτομέρειες του οικονομικού έτους**, απαντώντας στο ερώτημα «ποια είναι η τρέχουσα ρύθμιση οικονομικού έτους;».

### Βήμα 3: Ορισμός Έναρξης Οικονομικού Έτους και Ενεργοποίηση Αρίθμησης
```java
//Create a project instance
Project prj = new Project();
//Set fiscal year properties
prj.set(Prj.FY_START_DATE, Month.JULY);
prj.set(Prj.FISCAL_YEAR_START, new NullableBool(true));
//Save the project as XML project file
prj.save(dataDir + "savedProject.xml", SaveFileFormat.Xml);
```
Σε αυτό το βήμα δείχνουμε πώς να ορίσετε τις ρυθμίσεις του οικονομικού έτους:
- `Month.JULY` ορίζει τον μήνα έναρξης του οικονομικού έτους.  
- `NullableBool(true)` ενεργοποιεί την αρίθμηση του οικονομικού έτους.  
- Τέλος, το έργο **μετατρέπεται από MPP σε XML** χρησιμοποιώντας το `SaveFileFormat.Xml`.

### Βήμα 4: Επιβεβαίωση Λειτουργίας
```java
//Display result of conversion.
System.out.println("Process completed Successfully");
```
Ένα απλό μήνυμα στην κονσόλα επιβεβαιώνει ότι το οικονομικό έτος έχει ρυθμιστεί και το αρχείο αποθηκεύτηκε.
