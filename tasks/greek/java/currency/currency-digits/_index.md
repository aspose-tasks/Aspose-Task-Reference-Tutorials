# Πώς να Λάβετε το Νόμισμα από το MS Project χρησιμοποιώντας το Aspose.Tasks

## Εισαγωγή
Αν αναρωτιέστε **πώς να λάβετε το νόμισμα** από ένα αρχείο Microsoft Project, βρίσκεστε στο σωστό μέρος. Σε αυτό το ολοκληρωμένο tutorial θα ανακαλύψετε **πώς να εργαστείτε με ms project currency** τιμές χρησιμοποιώντας τη βιβλιοθήκη Aspose.Tasks για Java. Είτε δημιουργείτε ένα εργαλείο αναφοράς, ένα εργαλείο μετεγκατάστασης, ή απλώς χρειάζεστε να διαβάσετε τις ρυθμίσεις νομίσματος από ένα **java project file**, αυτός ο οδηγός σας καθοδηγεί βήμα προς βήμα—από τη φόρτωση ενός αρχείου *.mpp* μέχρι την εξαγωγή των ψηφίων του νομίσματος. Στο τέλος, θα είστε άνετοι με τη διαχείριση δεδομένων ms project currency στις δικές σας εφαρμογές.

## Γρήγορες Απαντήσεις
- **Ποια βιβλιοθήκη διαβάζει αρχεία MS Project;** Aspose.Tasks for Java.  
- **Πόσες γραμμές κώδικα χρειάζονται για να λάβετε τα ψηφία του νομίσματος;** Μόνο τρεις σύντομες γραμμές μετά τη φόρτωση του έργου.  
- **Χρειάζομαι άδεια για ανάπτυξη;** Μια δωρεάν δοκιμή λειτουργεί για δοκιμές· απαιτείται εμπορική άδεια για παραγωγή.  
- **Ποια έκδοση της Java υποστηρίζεται;** Java 8 ή νεότερη (οποιοδήποτε JDK τρέχει το Aspose.Tasks).  
- **Μπορώ να ανακτήσω άλλες ιδιότητες του Project;** Ναι – το Aspose.Tasks εκθέτει ένα πλήρες σύνολο πεδίων Project (π.χ., ημερομηνία έναρξης, τιμές κόστους κ.λπ.).

## Τι είναι το ms project currency;
`ms project currency` αναφέρεται στην αριθμητική ακρίβεια (αριθμός δεκαδικών θέσεων) που χρησιμοποιεί το Microsoft Project όταν εμφανίζει χρηματικές τιμές. Αποθηκεύεται στο αρχείο Project ως η ιδιότητα **CURRENCY_DIGITS** και καθορίζει αν τα ποσά εμφανίζονται ως ακέραιοι, με μία δεκαδική, δύο δεκαδικές κ.λπ.

## Γιατί να χρησιμοποιήσετε το Aspose.Tasks για τη διαχείριση του ms project currency;
- **Δεν απαιτείται εγκατάσταση του Microsoft Project** – εργασία απευθείας με αρχεία *.mpp* σε οποιαδήποτε πλατφόρμα που υποστηρίζει Java.  
- **Ισχυρή ασφάλεια τύπων** – το API επιστρέφει τιμές με ισχυρό τύπο, μειώνοντας τα σφάλματα ανάλυσης.  
- **Βελτιστοποιημένη απόδοση** – φορτώνει μεγάλα έργα γρήγορα και εξάγει μόνο τα πεδία που χρειάζεστε.  
- **Διαπλατφορμική** – εκτελείται σε Windows, Linux ή macOS χωρίς τροποποίηση.

## Προαπαιτούμενα
Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε τα εξής:

1. **Περιβάλλον Ανάπτυξης Java** – εγκατεστημένο και ρυθμισμένο JDK 8 ή νεότερο.  
2. **Aspose.Tasks for Java** – κατεβάστε το τελευταίο JAR από την επίσημη ιστοσελίδα: [Aspose.Tasks for Java](https://releases.aspose.com/tasks/java/).  
3. **Βασικές γνώσεις Java** – πρέπει να είστε άνετοι με τη δημιουργία ενός έργου Java, την προσθήκη εξωτερικών βιβλιοθηκών και την εκτέλεση μιας μεθόδου `main`.  

## Εισαγωγή Πακέτων
Πρώτα, εισάγετε τις κλάσεις που θα χρειαστούμε.  
```java
import java.io.IOException;
import com.aspose.tasks.Prj;
import com.aspose.tasks.Project;
import com.aspose.tasks.SaveFileFormat;
```

## Βήμα 1: Ορισμός Καταλόγου Δεδομένων
Καθορίστε το φάκελο που περιέχει το **java project file** (`*.mpp`).  
```java
String dataDir = "Your Data Directory";
```
Αντικαταστήστε `"Your Data Directory"` με την απόλυτη ή σχετική διαδρομή όπου βρίσκεται το `project.mpp`.

## Βήμα 2: Φόρτωση του Αρχείου MPP  
Τώρα θα δούμε **πώς να φορτώσετε αρχεία mpp** χρησιμοποιώντας το Aspose.Tasks.  
```java
Project project = new Project(dataDir + "project.mpp");
```
Βεβαιωθείτε ότι το όνομα του αρχείου ταιριάζει ακριβώς· διαφορετικά, θα προκληθεί `IOException`.

## Βήμα 3: Ανάκτηση των Ψηφίων Νομίσματος  
Με το έργο φορτωμένο, η εξαγωγή των **ms project currency** ψηφίων είναι μια γραμμή κώδικα:  
```java
System.out.println(project.get(Prj.CURRENCY_DIGITS));
```
Η κλήση επιστρέφει ένα `Integer` που αντιπροσωπεύει τον αριθμό των δεκαδικών θέσεων (π.χ., `2` για λεπτά). Η τιμή εκτυπώνεται στην κονσόλα, αλλά μπορείτε επίσης να την αποθηκεύσετε σε μεταβλητή για περαιτέρω επεξεργασία.

## Κοινά Προβλήματα & Συμβουλές
- **Αρχείο δεν βρέθηκε** – ελέγξτε ξανά τη διαδρομή `dataDir` και βεβαιωθείτε ότι το όνομα του αρχείου είναι σωστό, συμπεριλαμβανομένης της επέκτασης `.mpp`.  
- **Μη υποστηριζόμενη έκδοση αρχείου** – το Aspose.Tasks υποστηρίζει μορφές Project 2000‑2024· παλαιότερα ή κατεστραμμένα αρχεία μπορεί να χρειάζονται μετατροπή.  
- **Δεν έχει οριστεί άδεια** – κατά την ανάπτυξη μια δοκιμαστική άδεια λειτουργεί, αλλά για παραγωγή πρέπει να εφαρμόσετε έγκυρη άδεια ώστε να αποφύγετε υδατογραφήματα αξιολόγησης.
