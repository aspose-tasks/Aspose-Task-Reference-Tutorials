# Διαχείριση ψηφίων νομίσματος του MS Project με Aspose.Tasks

## Εισαγωγή
Σε αυτό το ολοκληρωμένο tutorial θα ανακαλύψετε **πώς να εργάζεστε με τις τιμές νομίσματος του MS Project** χρησιμοποιώντας τη βιβλιοθήκη Aspose.Tasks για Java. Είτε δημιουργείτε ένα εργαλείο αναφορών, ένα βοηθητικό πρόγραμμα μετεγκατάστασης, είτε απλώς χρειάζεστε να διαβάσετε τις ρυθμίσεις νομίσματος από ένα **αρχείο java project**, αυτός ο οδηγός σας καθοδηγεί βήμα προς βήμα—από τη φόρτωση ενός αρχείου *.mpp* μέχρι την εξαγωγή των ψηφίων νομίσματος. Στο τέλος, θα είστε άνετοι με τη διαχείριση δεδομένων νομίσματος του MS Project στις δικές σας εφαρμογές.

## Γρήγορες Απαντήσεις
- **Ποια βιβλιοθήκη διαβάζει αρχεία MS Project;** Aspose.Tasks for Java.  
- **Πόσες γραμμές κώδικα χρειάζονται για να ληφθούν τα ψηφία νομίσματος;** Μόνο τρεις σύντομες γραμμές μετά τη φόρτωση του έργου.  
- **Χρειάζεται άδεια για ανάπτυξη;** Μια δωρεάν δοκιμή λειτουργεί για δοκιμές· απαιτείται εμπορική άδεια για παραγωγή.  
- **Ποια έκδοση Java υποστηρίζεται;** Java 8 ή νεότερη (οποιοδήποτε JDK που τρέχει το Aspose.Tasks).  
- **Μπορώ να ανακτήσω άλλες ιδιότητες του Project;** Ναι – το Aspose.Tasks εκθέτει ένα πλήρες σύνολο πεδίων Project (π.χ., ημερομηνία έναρξης, τιμές κόστους κ.λπ.).

## Τι είναι το νόμισμα του MS Project;
`ms project currency` αναφέρεται στην αριθμητική ακρίβεια (αριθμός δεκαδικών θέσεων) που χρησιμοποιεί το Microsoft Project όταν εμφανίζει χρηματικές τιμές. Αποθηκεύεται στο αρχείο Project ως η ιδιότητα **CURRENCY_DIGITS** και καθορίζει εάν τα ποσά εμφανίζονται ως ακέραιοι, με μία δεκαδική, δύο δεκαδικές κ.λπ.

## Γιατί να χρησιμοποιήσετε το Aspose.Tasks για τη διαχείριση του νομίσματος του MS Project;
- **Δεν απαιτείται εγκατάσταση Microsoft Project** – εργάζεστε απευθείας με αρχεία *.mpp* σε οποιαδήποτε πλατφόρμα υποστηρίζει Java.  
- **Ισχυρή τυπική ασφάλεια** – το API επιστρέφει ισχυρά τυποποιημένες τιμές, μειώνοντας τα σφάλματα ανάλυσης.  
- **Βελτιστοποιημένη απόδοση** – φορτώνει μεγάλα έργα γρήγορα και εξάγει μόνο τα πεδία που χρειάζεστε.  
- **Διαπλατφορμική** – τρέχει σε Windows, Linux ή macOS χωρίς τροποποίησηαπαιτούμενα
Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε τα εξής:

1. **Περιβάλλον Ανάπτυξης Java** – εγκατεστημένο και ρυθμισμένο JDK 8 ή νεότερο.  
2. **Aspose.Tasks for Java** – κατεβάστε το τελευταίο JAR από την επίσημη ιστοσελίδα: [Aspose.Tasks for Java](https://releases.aspose.com/tasks/java/).  
3. **Βασικές γνώσεις Java** – πρέπει να μπορείτε να δημιουργήσετε ένα έργο Java, να προσθέσετε εξωτερικές βιβλιοθήκες και να εκτελέσετε μια `main` μέθοδο.  

## Εισαγωγή Πακέτων
Πρώτα, εισάγετε τις κλάσεις που θα χρειαστούμε.  
```java
import java.io.IOException;
import com.aspose.tasks.Prj;
import com.aspose.tasks.Project;
import com.aspose.tasks.SaveFileFormat;
```

## Βήμα 1: Ορισμός Καταλόγου Δεδομένων
Καθορίστε το φάκελο που περιέχει το **αρχείο java project** (`*.mpp`).  
```java
String dataDir = "Your Data Directory";
```
Αντικαταστήστε το `"Your Data Directory"` με το απόλυτο ή σχετικό μονοπάτι όπου βρίσκεται το `project.mpp`.

## Βήμα 2: Φόρτωση του Αρχείου MPP  
Τώρα θα δούμε **πώς να φορτώνουμε αρχεία mpp** χρησιμοποιώντας το Aspose.Tasks.  
```java
Project project = new Project(dataDir + "project.mpp");
```
Βεβαιωθείτε ότι το όνομα του αρχείου ταιριάζει ακριβώς· διαφορετικά θα προκληθεί `IOException`.

## Βήμα 3: Ανάκτηση Ψηφίων Νομίσματος  
Με το έργο φορτωμένο, η εξαγωγή των **ψηφίων νομίσματος του ms project** είναι μια γραμμή κώδικα:  
```java
System.out.println(project.get(Prj.CURRENCY_DIGITS));
```
Η κλήση επιστρέφει ένα `Integer` που αντιπροσωπεύει τον αριθμό των δεκαδικών θέσεων (π.χ., `2` για σεντς). Η τιμή εκτυπώνεται στην κονσόλα, αλλά μπορείτε επίσης να την αποθηκεύσετε σε μεταβλητή για περαιτέρω επεξεργασία.

## Συχνά Προβλήματα & Συμβουλές
- **Αρχείο δεν βρέθηκε** – ελέγξτε ξανά τη διαδρομή `dataDir` και βεβαιωθείτε ότι το όνομα του αρχείου είναι σωστό, συμπεριλαμβανομένης της επέκτασης `.mpp`.  
- **Μη υποστηριζόμενη έκδοση αρχείου** – το Aspose.Tasks υποστηρίζει μορφές Project 2000‑2024· παλαιότερα ή κατεστραμμένα αρχεία μπορεί να χρειάζονται μετατροπή.  
- **Δεν έχει οριστεί άδεια** – κατά την ανάπτυξη η δοκιμαστική έκδοση λειτουργεί, αλλά για παραγωγή πρέπει να εφαρμόσετε έγκυρη άδεια ώστε να αποφύγετε υδατογραφήματα αξιολόγησης.
