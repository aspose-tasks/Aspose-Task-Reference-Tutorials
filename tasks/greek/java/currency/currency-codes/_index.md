# Πώς να διαβάσετε αρχείο MS Project και να διαχειριστείτε κωδικούς νομισμάτων με το Aspose.Tasks

## Introduction
Καλώς ήρθατε! Σε αυτό το tutorial θα ανακαλύψετε **πώς να διαβάσετε δεδομένα αρχείου MS Project** και συγκεκριμένα **πώς να διαχειριστείτε κωδικούς νομισμάτων** χρησιμοποιώντας το Aspose.Tasks Java API. Είτε δημιουργείτε αναφορές, ενοποιείτε έργα πολλαπλών νομισμάτων, ή απλώς χρειάζεστε να εξασφαλίσετε ότι εμφανίζονται τα σωστά χρηματοοικονομικά σύμβολα, αυτός ο οδηγός σας καθοδηγεί βήμα προς βήμα—από τη ρύθμιση του περιβάλλοντος μέχρι την ανάκτηση του κωδικού νομίσματος προγραμματιστικά. Στο τέλος, θα είστε άνετοι με το διάβασμα αρχείων MS Project και την εξαγωγή των πληροφοριών νομίσματος που χρειάζεστε.

## Quick Answers
- **Τι κάνει το API;** Διαβάζει αρχεία MS Project και εκθέτει ιδιότητες όπως ο κωδικός νομίσματος.  
- **Ποια γλώσσα χρησιμοποιείται;** Java, μέσω της βιβλιοθήκης Aspose.Tasks for Java.  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για ανάπτυξη· απαιτείται εμπορική άδεια για παραγωγή.  
- **Μπορώ να ανακτήσω τον κωδικό σε μία γραμμή;** Ναι—`prj.get(Prj.CURRENCY_CODE)` επιστρέφει τη συμβολοσειρά του κωδικού νομίσματος.  
- **Είναι συμβατό με όλες τις εκδόσεις του Project;** Το Aspose.Tasks υποστηρίζει τόσο παλαιότερες (MPP) όσο και νεότερες (XML, XER) μορφές.

## What is **read ms project file**?
Το διάβασμα ενός αρχείου MS Project σημαίνει το προγραμματιστικό άνοιγμα ενός *.mpp* (ή άλλου υποστηριζόμενου) εγγράφου έργου και η πρόσβαση στις δομές δεδομένων του—εργασίες, πόροι, ημερολόγια και ρυθμίσεις χρηματοοικονομικών—χωρίς χειροκίνητη αλληλεπίδραση με το Microsoft Project.

## Why use Aspose.Tasks to **read msproject** files?
- **Πλήρης υποστήριξη μορφών** – λειτουργεί με αρχεία από το Project 98 μέχρι τις πιο πρόσφατες εκδόσεις του Office.  
- **Δεν απαιτείται εγκατάσταση COM ή Office** – καθαρή Java, ιδανική για αυτοματοποίηση στο διακομιστή.  
- **Πλούσιο API** – παρέχει άμεση πρόσβαση σε ιδιότητες όπως `Prj.CURRENCY_CODE`, επιτρέποντας να **how to retrieve currency** πληροφορίες άμεσα.  
- **Απόδοση** – ελαφρύ parsing, ιδανικό για επεξεργασία μεγάλου αριθμού έργων σε batch.

## Prerequisites
Πριν βουτήξουμε στον κώδικα, βεβαιωθείτε ότι έχετε τα παρακάτω:

### Java Development Kit (JDK) Installed
Βεβαιωθείτε ότι έχετε εγκατεστημένο πρόσφατο JDK στο σύστημά σας. Μπορείτε να το κατεβάσετε και να το εγκαταστήσετε από [εδώ](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html).

### Aspose.Tasks for Java Library
Κατεβάστε και ρυθμίστε τη βιβλιοθήκη Aspose.Tasks for Java. Αναλυτική τεκμηρίωση και τα πιο πρόσφατα binaries είναι διαθέσιμα [εδώ](https://reference.aspose.com/tasks/java/).

## Import Packages
Για να ξεκινήσετε, ας εισάγουμε τα απαραίτητα πακέτα στο Java project σας:
```java
import com.aspose.tasks.Prj;
import com.aspose.tasks.Project;
```

## Step‑by‑step guide

### Step 1: Set Up Data Directory
Ορίστε το φάκελο που περιέχει το *.mpp* αρχείο σας. Προσαρμόστε τη διαδρομή ώστε να ταιριάζει με το περιβάλλον σας.
```java
String dataDir = "Your Data Directory";
```

### Step 2: Load the Project File
Δημιουργήστε ένα αντικείμενο `Project` φορτώνοντας το αρχείο MS Project. Αυτό είναι το σημείο όπου **read msproject** δεδομένα φορτώνονται στη μνήμη.
```java
Project prj = new Project(dataDir + "project.mpp");
```

### Step 3: Retrieve Currency Code
Τώρα που το έργο έχει φορτωθεί, μπορείτε να **how to retrieve currency** πληροφορίες με μία κλήση. Αυτό δείχνει τη χρήση του **get currency code java**.
```java
System.out.println(prj.get(Prj.CURRENCY_CODE));
```
Η έξοδος θα είναι ο τριψήφιος κωδικός ISO του νομίσματος (π.χ., `USD`, `EUR`, `GBP`) που έχει ρυθμιστεί στο έργο.

### Step 4: (Optional) Use the Currency Code
Μπορεί να θέλετε να χρησιμοποιήσετε τον κωδικό που ανακτήθηκε αλλού—όπως για τη μορφοποίηση τιμών κόστους σε προσαρμοσμένη αναφορά ή για να τον περάσετε σε χρηματοοικονομικό API. Εδώ είναι μια γρήγορη εικονογράφηση (δεν απαιτείται επιπλέον μπλοκ κώδικα):
- Αποθηκεύστε τον κωδικό σε μια μεταβλητή.  
- Χρησιμοποιήστε τον όταν δημιουργείτε συμβολοσειρές χρηματικών ποσών.  
- Περάστε τον σε υπηρεσίες downstream που απαιτούν αναγνωριστικό νομίσματος.

## Common Issues and Solutions
| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| **Null output** | Το αρχείο έργου δεν ορίζει νόμισμα (η προεπιλογή είναι κενή). | Ορίστε το νόμισμα στο Microsoft Project ή εκχωρήστε το μέσω `prj.set(Prj.CURRENCY_CODE, "USD");` πριν το διαβάσετε. |
| **File not found** | Λανθασμένη διαδρομή `dataDir`. | Επαληθεύστε τη διαδρομή και βεβαιωθείτε ότι το όνομα αρχείου ταιριάζει ακριβώς, συμπεριλαμβανομένης της ευαισθησίας σε πεζά/κεφαλαία. |
| **Unsupported file version** | Πολύ παλιό ή κατεστραμμένο αρχείο *.mpp*. | Χρησιμοποιήστε την πιο πρόσφατη έκδοση του Aspose.Tasks ή μετατρέψτε το αρχείο σε νεότερη μορφή στο Microsoft Project πρώτα. |
