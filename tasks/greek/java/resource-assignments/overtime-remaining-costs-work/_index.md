# Παρακολούθηση Κόστους Έργου με Aspose.Tasks: Πρόσθετες Ώρες & Εργασία

## Εισαγωγή
Σε αυτό το σεμινάριο θα ανακαλύψετε πώς να **παρακολουθείτε το κόστος του έργου** χρησιμοποιώντας το Aspose.Tasks για Java. Θα περάσουμε από τη διαδικασία παρακολούθησης υπερωριών, εναπομείναντος κόστους και εργασίας ώστε να διατηρείτε τα έργα σας εντός χρονοδιαγράμματος και προϋπολογισμού. Είτε είστε διαχειριστής έργου είτε αρχηγός ομάδας, αυτά τα βήματα θα σας βοηθήσουν να διατηρήσετε σαφή ορατότητα στα οικονομικά και τους πόρους.

## Τι είναι η παρακολούθηση κόστους έργου;
Η παρακολούθηση κόστους έργου είναι η πρακτική της συνεχούς παρακολούθησης όλων των οικονομικών πτυχών ενός έργου — προϋπολογισμένων εξόδων, πραγματικών δαπανών και προβλεπόμενων εναπομείναντων εξόδων. Ενσωματώνοντας αυτό με τις αναθέσεις πόρων, αποκτάτε άμεση εικόνα για τα έξοδα υπερωριών και την πρόοδο της εργασίας.

## Γιατί να παρακολουθείτε τις υπερωρίες και την εναπομείναν εργασία;
- **Έλεγχος προϋπολογισμών:** Οι υπερωρίες συχνά προκαλούν απρόσμενες υπερβάσεις κόστους.  
- **Βελτίωση πρόβλεψης:** Η γνώση της εναπομείναν εργασίας σας βοηθά να προσαρμόζετε τα χρονοδιαγράμματα προληπτικά.  
- **Αύξηση διαφάνειας:** Τα ενδιαφερόμενα μέρη μπορούν να δουν ακριβώς πού καταναλώνονται οι πόροι.

## Εισαγωγή Πακέτων
Ξεκινήστε εισάγοντας τα απαραίτητα πακέτα στο αρχείο Java σας:
```java
import com.aspose.tasks.Asn;
import com.aspose.tasks.Project;
import com.aspose.tasks.ResourceAssignment;
```

## Βήμα 1: Ρύθμιση του Καταλόγου Δεδομένων
Ορίστε τον κατάλογο όπου βρίσκεται το αρχείο του έργου σας:
```java
String dataDir = "Your Data Directory";
```
Αντικαταστήστε το `"Your Data Directory"` με τη διαδρομή του αρχείου του έργου σας.

## Βήμα 2: Φόρτωση του Έργου
Δημιουργήστε ένα αντικείμενο `Project` και φορτώστε το αρχείο του έργου:
```java
Project project = new Project(dataDir + "ResourceAssignmentOvertimes.mpp");
```
Αντικαταστήστε το `"ResourceAssignmentOvertimes.mpp"` με το όνομα του αρχείου MPP σας. Αυτό το βήμα δείχνει τη χρήση της **φόρτωσης αρχείου mpp**.

## Βήμα 3: Επανάληψη Στις Αναθέσεις Πόρων
Επανάληψη σε κάθε ανάθεση πόρου στο έργο:
```java
for (ResourceAssignment ra : project.getResourceAssignments()) {
```

## Βήμα 4: Εκτύπωση Κόστους Υπερωριών και Εργασίας
Ανάκτηση και εκτύπωση του κόστους υπερωριών και της εργασίας για κάθε ανάθεση πόρου:
```java
    System.out.println(ra.get(Asn.OVERTIME_COST));
    System.out.println(ra.get(Asn.OVERTIME_WORK).toString());
```

## Βήμα 5: Εκτύπωση Εναπομείναντος Κόστους και Εργασίας
Ανάκτηση και εκτύπωση του εναπομείναντος κόστους και της εργασίας για κάθε ανάθεση πόρου:
```java
    System.out.println(ra.get(Asn.REMAINING_COST));
    System.out.println(ra.get(Asn.REMAINING_WORK).toString());
```

## Βήμα 6: Εκτύπωση Εναπομείναντος Κόστους Υπερωριών και Εργασίας
Ανάκτηση και εκτύπωση του εναπομείναντος κόστους υπερωριών και της εργασίας για κάθε ανάθεση πόρου:
```java
    System.out.println(ra.get(Asn.REMAINING_OVERTIME_COST));
    System.out.println(ra.get(Asn.REMAINING_OVERTIME_WORK).toString());
}
```

## Συχνά Προβλήματα και Λύσεις
- **Αρχείο δεν βρέθηκε:** Ελέγξτε ξανά τη διαδρομή `dataDir` και βεβαιωθείτε ότι το όνομα του αρχείου MPP είναι σωστό.  
- **Τιμές null:** Ορισμένες αναθέσεις μπορεί να μην έχουν δεδομένα υπερωριών· προστατέψτε τον κώδικα από `null` κατά την εκτύπωση.  
- **Ασυμφωνία έκδοσης:** Χρησιμοποιήστε μια έκδοση της βιβλιοθήκης που ταιριάζει με τη μορφή του αρχείου MPP (π.χ., νεότερες εκδόσεις του MS Project).
