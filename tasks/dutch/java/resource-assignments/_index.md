## Frequently Asked Questions

**Q: Can I use Aspose.Tasks to update budgets in an existing MS Project file?**  
A: Yes, the API lets you open, modify the Budget field of any assignment, and save the changes without needing MS Project installed.

**Q: Is there a limit to the number of assignments I can manage?**  
A: Aspose.Tasks handles thousands of assignments efficiently; performance depends mainly on available memory.

**Q: Do I need to handle currency conversion manually?**  
A: The library works with the numeric budget value; you’ll need to apply any currency conversion in your own code.

**Q: How do I read the budget of a specific resource assignment?**  
A: Load the project, locate the Assignment by its ID or Resource UID, then use `assignment.getBudget()` (Java method) to retrieve the value.

**Q: What version of Aspose.Tasks is required for budget management?**  
A: All recent releases (2023‑2025) support budget properties; we recommend using the latest stable version for best compatibility.
