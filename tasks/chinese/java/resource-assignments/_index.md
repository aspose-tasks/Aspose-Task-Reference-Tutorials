## 常见问题

**Q: 我可以使用 Aspose.Tasks 更新现有 MS Project 文件中的预算吗？**  
A: 可以，API 允许您打开文件、修改任意分配的 Budget 字段，并在无需安装 MS Project 的情况下保存更改。

**Q: 我可以管理的分配数量有限制吗？**  
A: Aspose.Tasks 能高效处理成千上万的分配，性能主要受可用内存影响。

**Q: 我需要手动处理货币转换吗？**  
A: 库仅处理数值预算，货币转换需在您的代码中自行实现。

**Q: 如何读取特定资源分配的预算？**  
A: 加载项目后，根据 ID 或 Resource UID 定位 Assignment，然后调用 `assignment.getBudget()`（Java 方法）获取数值。

**Q: 管理预算需要哪个版本的 Aspose.Tasks？**  
A: 所有近期发布（2023‑2025）的版本均支持预算属性；建议使用最新稳定版以获得最佳兼容性。
