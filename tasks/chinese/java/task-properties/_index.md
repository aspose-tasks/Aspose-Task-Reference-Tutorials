## 常见问题

**Q: 我需要单独的库来处理任务优先级吗？**  
A: 不需要。Aspose.Tasks Java API 已完整支持任务优先级，您可以直接在 `Task` 对象上设置。

**Q: 我可以导入已有的 .mpp 文件，修改任务后再保存吗？**  
A: 可以。使用 `Project project = new Project("input.mpp");` 修改任务后，调用 `project.save("output.mpp");`。

**Q: 能否一次性批量更新大量任务？**  
A: 完全可以。遍历 `project.getRootTask().getChildren()` 并在循环中应用更改即可实现批量操作。

**Q: 如何处理自定义扩展属性？**  
A: 创建 `ExtendedAttributeDefinition`，将其添加到项目中，然后通过 `task.getExtendedAttributes().add(...)` 为每个任务分配值。

**Q: Java 11 需要哪个版本的 Aspose.Tasks？**  
A: 2022 年之后发布的任何 Aspose.Tasks 版本都完整支持 Java 11 及更高版本。
