## 常见问题

**Q: 如何使用 Aspose.Tasks 开始更新 MPP 文件？**  
A: 使用 `new Project("file.mpp")` 加载项目，修改所需对象（任务、资源等），然后调用 `project.save("file.mpp", SaveFileFormat.MPP)` 保存。

**Q: 更新任务后我可以计算关键路径吗？**  
A: 可以——在进行更改后使用 `project.getCriticalPath()`；API 会自动重新计算路径。

**Q: 能否读取 Primavera XML 并随后更新生成的 MPP？**  
A: 完全可以。使用 `new Project("primavera.xml")` 导入 Primavera 文件，进行更新后保存为 MPP。

**Q: 处理扩展属性时需要注意什么？**  
A: 在赋值前确保属性定义已存在；否则，API 将抛出 `ArgumentException`。

**Q: 推荐使用哪个版本的 Aspose.Tasks 来学习这些教程？**  
A: 推荐使用最新的稳定版本（例如 Aspose.Tasks 24.12），以获得错误修复和新功能的优势。
