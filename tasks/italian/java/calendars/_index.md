## Domande frequenti

**Q: Come creo programmaticamente **how to create calendar** oggetti in Aspose.Tasks?**  
A: Istanzia `com.aspose.tasks.Calendar`, aggiungilo a un `Project` tramite `project.getCalendars().add(calendar)`, quindi configura i suoi `WeekDays`, `Exceptions` e `WorkingTimes`.

**Q: Qual è il modo migliore per **how to extract hours** da un calendario esistente?**  
A: Usa la collezione `Calendar.getWorkingTimes()`; ogni voce `WorkingTime` contiene gli orari di inizio e fine per il giorno specificato.

**Q: Posso **how to define weekdays** per una settimana lavorativa personalizzata?**  
A: Sì. Recupera gli oggetti `WeekDay` dal calendario (`calendar.getWeekDays()`) e imposta `DayWorking` e `WorkingTimes` per ciascun giorno che desideri personalizzare.

**Q: Esiste un metodo semplice per **how to update mpp** file dopo aver modificato un calendario?**  
A: Dopo aver modificato il calendario, chiama `project.save("UpdatedProject.mpp", SaveFileFormat.MPP)` per scrivere le modifiche nel file MPP.

**Q: È necessaria una licenza separata per ogni operazione sul calendario?**  
A: No. Una singola licenza Aspose.Tasks copre tutte le API relative ai calendari, inclusa la creazione, l'estrazione, la definizione e gli aggiornamenti MPP.
