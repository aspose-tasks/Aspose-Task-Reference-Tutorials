# Aspose.Tasks के साथ MS Project फ़ाइल पढ़ना और मुद्रा कोड प्रबंधित करना

## परिचय
स्वागत है! इस ट्यूटोरियल में आप **MS Project फ़ाइल** डेटा को पढ़ना और विशेष रूप से **मुद्रा कोड** को Aspose.Tasks Java API का उपयोग करके प्रबंधित करना सीखेंगे। चाहे आप रिपोर्ट बना रहे हों, बहु‑मुद्रा प्रोजेक्ट को समेकित कर रहे हों, या केवल यह सुनिश्चित करना चाहते हों कि सही वित्तीय प्रतीक दिखें, यह गाइड आपको पर्यावरण सेटअप से लेकर प्रोग्रामेटिक रूप से मुद्रा कोड प्राप्त करने तक हर कदम पर मार्गदर्शन करेगा। अंत तक, आप MS Project फ़ाइलें पढ़ने और आवश्यक मुद्रा जानकारी निकालने में सहज हो जाएंगे।

## त्वरित उत्तर
- **API क्या करता है?** यह MS Project फ़ाइलें पढ़ता है और मुद्रा कोड जैसी प्रॉपर्टीज़ को उजागर करता है।  
- **कौन सी भाषा उपयोग की गई है?** Java, Aspose.Tasks for Java लाइब्रेरी के माध्यम से।  
- **क्या लाइसेंस की आवश्यकता है?** विकास के लिए एक मुफ्त ट्रायल काम करता है; उत्पादन के लिए एक व्यावसायिक लाइसेंस आवश्यक है।  
- **क्या मैं कोड एक ही लाइन में प्राप्त कर सकता हूँ?** हाँ—`prj.get(Prj.CURRENCY_CODE)` मुद्रा कोड स्ट्रिंग लौटाता है।  
- **क्या यह सभी Project संस्करणों के साथ संगत है?** Aspose.Tasks पुराने (MPP) और नए (XML, XER) दोनों फ़ॉर्मैट्स को सपोर्ट करता है।

## **read ms project file** क्या है?
MS Project फ़ाइल पढ़ना मतलब प्रोग्रामेटिक रूप से *.mpp* (या अन्य समर्थित) प्रोजेक्ट दस्तावेज़ को खोलना और उसके डेटा संरचनाओं—टास्क, रिसोर्स, कैलेंडर, और वित्तीय सेटिंग्स—तक पहुँच प्राप्त करना, बिना Microsoft Project के मैनुअल इंटरैक्शन के।

## Aspose.Tasks के साथ **read msproject** फ़ाइलें क्यों उपयोग करें?
- **पूर्ण फ़ॉर्मैट समर्थन** – Project 98 से लेकर नवीनतम Office रिलीज़ तक की फ़ाइलों के साथ काम करता है।  
- **कोई COM या Office इंस्टॉलेशन आवश्यक नहीं** – शुद्ध Java, सर्वर‑साइड ऑटोमेशन के लिए आदर्श।  
- **समृद्ध API** – `Prj.CURRENCY_CODE` जैसी प्रॉपर्टीज़ तक सीधा एक्सेस देता है, जिससे आप **how to retrieve currency** जानकारी तुरंत प्राप्त कर सकते हैं।  
- **प्रदर्शन** – हल्का पार्सिंग, कई प्रोजेक्ट्स के बैच प्रोसेसिंग के लिए उपयुक्त।

## पूर्वापेक्षाएँ
कोड में डुबकी लगाने से पहले सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

### Java Development Kit (JDK) स्थापित है
सुनिश्चित करें कि आपके मशीन पर एक हालिया JDK उपलब्ध है। आप नवीनतम JDK संस्करण [यहाँ](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html) से डाउनलोड और इंस्टॉल कर सकते हैं।

### Aspose.Tasks for Java लाइब्रेरी
Aspose.Tasks for Java लाइब्रेरी डाउनलोड और सेट अप करें। विस्तृत दस्तावेज़ीकरण और नवीनतम बाइनरीज़ [यहाँ](https://reference.aspose.com/tasks/java/) उपलब्ध हैं।

## पैकेज इम्पोर्ट करें
शुरू करने के लिए, अपने Java प्रोजेक्ट में आवश्यक पैकेज इम्पोर्ट करें:
```java
import com.aspose.tasks.Prj;
import com.aspose.tasks.Project;
```

## चरण‑दर‑चरण गाइड

### चरण 1: डेटा डायरेक्टरी सेट करें
उस फ़ोल्डर को परिभाषित करें जिसमें आपकी *.mpp* फ़ाइल है। अपने पर्यावरण के अनुसार पाथ को समायोजित करें।
```java
String dataDir = "Your Data Directory";
```

### चरण 2: प्रोजेक्ट फ़ाइल लोड करें
MS Project फ़ाइल लोड करके एक `Project` इंस्टेंस बनाएं। यही वह बिंदु है जहाँ आप **read msproject** डेटा को मेमोरी में पढ़ते हैं।
```java
Project prj = new Project(dataDir + "project.mpp");
```

### चरण 3: मुद्रा कोड प्राप्त करें
अब प्रोजेक्ट लोड हो गया है, आप एक ही कॉल के साथ **how to retrieve currency** जानकारी प्राप्त कर सकते हैं। यह **get currency code java** उपयोग का प्रदर्शन करता है।
```java
System.out.println(prj.get(Prj.CURRENCY_CODE));
```
आउटपुट तीन‑अक्षर वाला ISO मुद्रा कोड होगा (जैसे `USD`, `EUR`, `GBP`) जो प्रोजेक्ट द्वारा उपयोग में लाया गया है।

### चरण 4: (वैकल्पिक) मुद्रा कोड का उपयोग करें
आप प्राप्त कोड को कहीं और लागू कर सकते हैं—जैसे कस्टम रिपोर्ट में लागत मानों को फॉर्मेट करना या इसे वित्तीय API को पास करना। यहाँ एक त्वरित उदाहरण है (कोई अतिरिक्त कोड ब्लॉक आवश्यक नहीं):
- कोड को एक वेरिएबल में स्टोर करें।  
- मौद्रिक स्ट्रिंग बनाते समय इसका उपयोग करें।  
- इसे उन डाउनस्ट्रीम सर्विसेज़ को पास करें जिन्हें मुद्रा पहचानकर्ता चाहिए।

## सामान्य समस्याएँ और समाधान
| समस्या | कारण | समाधान |
|-------|--------|-----|
| **Null आउटपुट** | प्रोजेक्ट फ़ाइल में मुद्रा परिभाषित नहीं है (डिफ़ॉल्ट खाली)। | Microsoft Project में मुद्रा सेट करें या पढ़ने से पहले `prj.set(Prj.CURRENCY_CODE, "USD");` के माध्यम से असाइन करें। |
| **फ़ाइल नहीं मिली** | `dataDir` पाथ गलत है। | पाथ को सत्यापित करें और सुनिश्चित करें कि फ़ाइल नाम बिल्कुल मेल खाता है, केस सेंसिटिविटी सहित। |
| **असमर्थित फ़ाइल संस्करण** | बहुत पुरानी या भ्रष्ट *.mpp* फ़ाइल। | नवीनतम Aspose.Tasks संस्करण का उपयोग करें या पहले Microsoft Project में फ़ाइल को नए फ़ॉर्मैट में परिवर्तित करें। |

## अक्सर पूछे जाने वाले प्रश्न

**प्रश्न: क्या Aspose.Tasks जटिल प्रोजेक्ट संरचनाओं को संभाल सकता है?**  
उत्तर: हाँ, API मल्टी‑लेवल टास्क हायरार्की, रिसोर्स पूल, और कस्टम फ़ील्ड्स को बिना समस्या के पढ़ और संशोधित कर सकता है।

**प्रश्न: क्या Aspose.Tasks विभिन्न संस्करणों की MS Project फ़ाइलों के साथ संगत है?**  
उत्तर: बिल्कुल। यह MPP, XML, XER, और कई Microsoft Project रिलीज़ के अन्य फ़ॉर्मैट्स को सपोर्ट करता है।

**प्रश्न: क्या Aspose.Tasks दस्तावेज़ीकरण और समर्थन प्रदान करता है?**  
उत्तर: व्यापक दस्तावेज़ीकरण, कोड उदाहरण, और Aspose वेबसाइट पर समर्पित समर्थन उपलब्ध है।

**प्रश्न: क्या मैं Aspose.Tasks को खरीदने से पहले आज़मा सकता हूँ?**  
उत्तर: एक मुफ्त ट्रायल उपलब्ध है जिससे आप सभी फीचर्स, जिसमें मुद्रा कोड पढ़ना भी शामिल है, का मूल्यांकन कर सकते हैं।

**प्रश्न: Aspose.Tasks के लिए अस्थायी लाइसेंस कहाँ प्राप्त कर सकते हैं?**  
उत्तर: अस्थायी लाइसेंस [वेबसाइट](https://purchase.aspose.com/temporary-license/) से छोटे‑समय के मूल्यांकन के लिए प्राप्त किए जा सकते हैं।
