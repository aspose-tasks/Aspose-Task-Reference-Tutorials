# Aspose.Tasks के साथ ms project मुद्रा अंकों को संभालें

## परिचय
इस व्यापक ट्यूटोरियल में आप **ms project मुद्रा के साथ काम करने का तरीका** Java के लिए Aspose.Tasks लाइब्रेरी का उपयोग करके सीखेंगे। चाहे आप रिपोर्टिंग टूल, माइग्रेशन यूटिलिटी बना रहे हों, या बस **java प्रोजेक्ट फ़ाइल** से मुद्रा सेटिंग्स पढ़नी हों, यह गाइड आपको हर कदम पर ले जाएगा—*.mpp* फ़ाइल लोड करने से लेकर मुद्रा अंकों को निकालने तक। अंत तक, आप अपने स्वयं के एप्लिकेशन में ms project मुद्रा डेटा को सहजता से संभालने में सक्षम हो जाएंगे।

## त्वरित उत्तर
- **MS Project फ़ाइलें पढ़ने वाली लाइब्रेरी कौन सी है?** Aspose.Tasks for Java.  
- **currency digits प्राप्त करने के लिए कितनी कोड लाइन्स चाहिए?** प्रोजेक्ट लोड होने के बाद केवल तीन संक्षिप्त लाइन्स.  
- **विकास के लिए मुझे लाइसेंस चाहिए?** परीक्षण के लिए मुफ्त ट्रायल काम करता है; उत्पादन के लिए व्यावसायिक लाइसेंस आवश्यक है.  
- **कौन सा Java संस्करण समर्थित है?** Java 8 या उससे ऊपर (कोई भी JDK जो Aspose.Tasks चलाता है).  
- **क्या मैं अन्य Project गुण प्राप्त कर सकता हूँ?** हां – Aspose.Tasks पूर्ण Project फ़ील्ड सेट (जैसे start date, cost rates, आदि) प्रदान करता है.

## ms project मुद्रा क्या है?
`ms project currency` वह संख्यात्मक सटीकता (दशमलव स्थानों की संख्या) दर्शाता है जो Microsoft Project मौद्रिक मानों को प्रदर्शित करते समय उपयोग करता है। यह Project फ़ाइल में **CURRENCY_DIGITS** प्रॉपर्टी के रूप में संग्रहीत होता है और निर्धारित करता है कि राशि पूर्णांक, एक‑दशमलव, दो‑दशमलव आदि के रूप में दिखेगी।

## ms project मुद्रा को संभालने के लिए Aspose.Tasks क्यों उपयोग करें?
- **Microsoft Project की स्थापना आवश्यक नहीं** – किसी भी प्लेटफ़ॉर्म पर सीधे *.mpp* फ़ाइलों के साथ काम करें जो Java का समर्थन करता है.  
- **मजबूत टाइप सुरक्षा** – API मजबूत‑टाइप्ड मान लौटाता है, जिससे पार्सिंग त्रुटियों में कमी आती है.  
- **प्रदर्शन‑उपयुक्त** – बड़े प्रोजेक्ट्स को तेज़ी से लोड करें और केवल आवश्यक फ़ील्ड निकालें.  
- **क्रॉस‑प्लेटफ़ॉर्म** – Windows, Linux, या macOS पर बिना संशोधन के चलाएँ.

## पूर्वापेक्षाएँ
शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

1. **Java विकास पर्यावरण** – JDK 8 या नया स्थापित और कॉन्फ़िगर किया हुआ.  
2. **Aspose.Tasks for Java** – आधिकारिक साइट से नवीनतम JAR डाउनलोड करें: [Aspose.Tasks for Java](https://releases.aspose.com/tasks/java/).  
3. **बुनियादी Java ज्ञान** – आपको Java प्रोजेक्ट बनाना, बाहरी लाइब्रेरी जोड़ना, और `main` मेथड चलाना आना चाहिए.  

## पैकेज आयात करें
पहले, उन क्लासों को आयात करें जिनकी हमें आवश्यकता होगी.  
```java
import java.io.IOException;
import com.aspose.tasks.Prj;
import com.aspose.tasks.Project;
import com.aspose.tasks.SaveFileFormat;
```

## चरण 1: डेटा डायरेक्टरी निर्धारित करें
उस फ़ोल्डर को निर्दिष्ट करें जिसमें आपकी **java प्रोजेक्ट फ़ाइल** (`*.mpp`) है.  
```java
String dataDir = "Your Data Directory";
```
`"Your Data Directory"` को उस पूर्ण या सापेक्ष पथ से बदलें जहाँ `project.mpp` स्थित है.

## चरण 2: MPP फ़ाइल लोड करें  
अब हम देखेंगे **Aspose.Tasks** का उपयोग करके **mpp फ़ाइलें कैसे लोड करें**.  
```java
Project project = new Project(dataDir + "project.mpp");
```
सुनिश्चित करें कि फ़ाइल नाम बिल्कुल मेल खाता है; अन्यथा, एक `IOException` फेंका जाएगा.

## चरण 3: मुद्रा अंकों को प्राप्त करें  
प्रोजेक्ट लोड होने के बाद, **ms project मुद्रा** अंकों को निकालना एक-लाइनर है:  
```java
System.out.println(project.get(Prj.CURRENCY_DIGITS));
```
यह कॉल एक `Integer` लौटाता है जो दशमलव स्थानों की संख्या दर्शाता है (उदाहरण के लिए, सेंट के लिए `2`)। मान को कंसोल में प्रिंट किया जाता है, लेकिन आप इसे आगे की प्रोसेसिंग के लिए वेरिएबल में भी स्टोर कर सकते हैं.

## सामान्य समस्याएँ और सुझाव
- **फ़ाइल नहीं मिली** – `dataDir` पथ को दोबारा जांचें और सुनिश्चित करें कि फ़ाइल नाम सही है, जिसमें `.mpp` एक्सटेंशन भी शामिल है.  
- **असमर्थित फ़ाइल संस्करण** – Aspose.Tasks Project 2000‑2024 फॉर्मैट्स का समर्थन करता है; पुराने या भ्रष्ट फ़ाइलों को रूपांतरण की आवश्यकता हो सकती है.  
- **लाइसेंस सेट नहीं है** – विकास के दौरान ट्रायल काम करता है, लेकिन उत्पादन में वैध लाइसेंस लागू करना आवश्यक है ताकि मूल्यांकन वॉटरमार्क न दिखें.

## अक्सर पूछे जाने वाले प्रश्न

**Q: क्या Aspose.Tasks मुद्रा अंकों के अलावा अन्य Project गुणों को संभाल सकता है?**  
A: हां, Aspose.Tasks विभिन्न कार्यों, संसाधनों और कस्टम फ़ील्ड्स जैसे Project फ़ाइलों के विभिन्न पहलुओं को बदलने के लिए व्यापक कार्यक्षमताएँ प्रदान करता है।

**Q: क्या Aspose.Tasks एंटरप्राइज़‑स्तर के अनुप्रयोगों के लिए उपयुक्त है?**  
A: बिल्कुल, Aspose.Tasks को एंटरप्राइज़‑ग्रेड प्रोजेक्ट्स की मांगों को पूरा करने के लिए डिज़ाइन किया गया है, जो उच्च प्रदर्शन और स्केलेबिलिटी प्रदान करता है।

**Q: क्या Aspose.Tasks क्रॉस‑प्लेटफ़ॉर्म विकास का समर्थन करता है?**  
A: हां, आप Aspose.Tasks for Java का उपयोग किसी भी प्लेटफ़ॉर्म पर कर सकते हैं जो Java Runtime Environment (Windows, Linux, macOS) का समर्थन करता है।

**Q: क्या मैं खरीदने से पहले Aspose.Tasks आज़मा सकता हूँ?**  
A: हां, आप [यहाँ](https://releases.aspose.com/) से मुफ्त ट्रायल संस्करण डाउनलोड कर सकते हैं।

**Q: मैं Aspose.Tasks के लिए समर्थन कहाँ प्राप्त कर सकता हूँ?**  
A: आप समर्थन [Aspose.Tasks फ़ोरम](https://forum.aspose.com/c/tasks/15) पर पा सकते हैं।
