# Aspose.Tasks का उपयोग करके MS Project से मुद्रा कैसे प्राप्त करें

## Introduction
यदि आप Microsoft Project फ़ाइल से **मुद्रा कैसे प्राप्त करें** जानकारी के बारे में सोच रहे हैं, तो आप सही जगह पर आए हैं। इस व्यापक ट्यूटोरियल में आप **ms project मुद्रा के साथ काम करना** Aspose.Tasks लाइब्रेरी for Java का उपयोग करके सीखेंगे। चाहे आप एक रिपोर्टिंग टूल, एक माइग्रेशन यूटिलिटी बना रहे हों, या बस **java प्रोजेक्ट फ़ाइल** से मुद्रा सेटिंग्स पढ़नी हों, यह गाइड आपको हर चरण से ले जाएगा—*.mpp* फ़ाइल लोड करने से लेकर मुद्रा अंकों को निकालने तक। अंत तक, आप अपने स्वयं के एप्लिकेशन में ms project मुद्रा डेटा को संभालने में सहज होंगे।

## Quick Answers
- **MS Project फ़ाइलें पढ़ने वाली लाइब्रेरी कौन सी है?** Aspose.Tasks for Java.  
- **मुद्रा अंकों को प्राप्त करने के लिए कितनी पंक्तियों का कोड चाहिए?** प्रोजेक्ट लोड होने के बाद केवल तीन संक्षिप्त पंक्तियाँ।  
- **क्या विकास के लिए लाइसेंस चाहिए?** परीक्षण के लिए एक मुफ्त ट्रायल काम करता है; उत्पादन के लिए एक व्यावसायिक लाइसेंस आवश्यक है।  
- **कौन सा Java संस्करण समर्थित है?** Java 8 या उससे ऊपर (कोई भी JDK जो Aspose.Tasks चलाता है)।  
- **क्या मैं अन्य प्रोजेक्ट गुण प्राप्त कर सकता हूँ?** हाँ – Aspose.Tasks प्रोजेक्ट फ़ील्ड्स का पूरा सेट प्रदान करता है (जैसे, प्रारंभ तिथि, लागत दरें, आदि)।

## What is ms project currency?
`ms project currency` वह संख्यात्मक सटीकता (दशमलव स्थानों की संख्या) को दर्शाता है जो Microsoft Project मौद्रिक मानों को प्रदर्शित करते समय उपयोग करता है। यह प्रोजेक्ट फ़ाइल में **CURRENCY_DIGITS** प्रॉपर्टी के रूप में संग्रहीत होता है और निर्धारित करता है कि राशि पूर्णांक, एक‑दशमलव, दो‑दशमलव आदि के रूप में दिखाई देगी।

## Why use Aspose.Tasks for handling ms project currency?
- **Microsoft Project इंस्टॉलेशन की आवश्यकता नहीं** – Java को सपोर्ट करने वाले किसी भी प्लेटफ़ॉर्म पर सीधे *.mpp* फ़ाइलों के साथ काम करें।  
- **मजबूत टाइप सुरक्षा** – API मजबूत‑टाइप्ड मान लौटाता है, जिससे पार्सिंग त्रुटियों में कमी आती है।  
- **प्रदर्शन‑ऑप्टिमाइज़्ड** – बड़े प्रोजेक्ट्स को जल्दी लोड करें और केवल आवश्यक फ़ील्ड्स निकालें।  
- **क्रॉस‑प्लेटफ़ॉर्म** – बिना संशोधन के Windows, Linux, या macOS पर चलाएँ।

## Prerequisites
शुरू करने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित हैं:

1. **Java विकास पर्यावरण** – JDK 8 या नया स्थापित और कॉन्फ़िगर किया हुआ।  
2. **Aspose.Tasks for Java** – आधिकारिक साइट से नवीनतम JAR डाउनलोड करें: [Aspose.Tasks for Java](https://releases.aspose.com/tasks/java/).  
3. **बुनियादी Java ज्ञान** – आपको Java प्रोजेक्ट बनाना, बाहरी लाइब्रेरी जोड़ना, और `main` मेथड चलाने में सहज होना चाहिए।  

## Import Packages
पहले, उन क्लासों को इम्पोर्ट करें जिनकी हमें आवश्यकता होगी।  
```java
import java.io.IOException;
import com.aspose.tasks.Prj;
import com.aspose.tasks.Project;
import com.aspose.tasks.SaveFileFormat;
```

## Step 1: Define Data Directory
उस फ़ोल्डर को निर्दिष्ट करें जिसमें आपकी **java प्रोजेक्ट फ़ाइल** (`*.mpp`) मौजूद है।  
```java
String dataDir = "Your Data Directory";
```
`"Your Data Directory"` को उस पूर्ण या सापेक्ष पथ से बदलें जहाँ `project.mpp` स्थित है।

## Step 2: Load the MPP File  
अब हम Aspose.Tasks का उपयोग करके **mpp फ़ाइलें कैसे लोड करें** देखेंगे।  
```java
Project project = new Project(dataDir + "project.mpp");
```
सुनिश्चित करें कि फ़ाइल नाम बिल्कुल मेल खाता हो; अन्यथा, एक `IOException` उत्पन्न होगा।

## Step 3: Retrieve Currency Digits  
प्रोजेक्ट लोड होने के बाद, **ms project मुद्रा** अंकों को निकालना एक पंक्ति में किया जा सकता है:  
```java
System.out.println(project.get(Prj.CURRENCY_DIGITS));
```
यह कॉल एक `Integer` लौटाता है जो दशमलव स्थानों की संख्या दर्शाता है (उदाहरण के लिए, सेंट के लिए `2`)। मान को कंसोल पर प्रिंट किया जाता है, लेकिन आप आगे की प्रोसेसिंग के लिए इसे एक वेरिएबल में भी संग्रहीत कर सकते हैं।

## Common Issues & Tips
- **फ़ाइल नहीं मिली** – `dataDir` पथ को दोबारा जांचें और सुनिश्चित करें कि फ़ाइल नाम सही है, जिसमें `.mpp` एक्सटेंशन भी शामिल है।  
- **असमर्थित फ़ाइल संस्करण** – Aspose.Tasks Project 2000‑2024 फ़ॉर्मैट्स को सपोर्ट करता है; पुराने या भ्रष्ट फ़ाइलों को रूपांतरण की आवश्यकता हो सकती है।  
- **लाइसेंस सेट नहीं है** – विकास के दौरान एक ट्रायल काम करता है, लेकिन उत्पादन के लिए आपको मूल्यांकन वॉटरमार्क से बचने हेतु एक वैध लाइसेंस लागू करना होगा।

## Frequently Asked Questions

**Q: क्या Aspose.Tasks मुद्रा अंकों के अलावा अन्य Project गुणों को संभाल सकता है?**  
**A:** हाँ, Aspose.Tasks प्रोजेक्ट फ़ाइलों के विभिन्न पहलुओं जैसे कार्य, संसाधन, और कस्टम फ़ील्ड्स को बदलने के लिए विस्तृत कार्यक्षमताएँ प्रदान करता है।

**Q: क्या Aspose.Tasks एंटरप्राइज़‑स्तर के अनुप्रयोगों के लिए उपयुक्त है?**  
**A:** बिल्कुल, Aspose.Tasks को एंटरप्राइज़‑ग्रेड प्रोजेक्ट्स की मांगों को पूरा करने के लिए डिज़ाइन किया गया है, जो उच्च प्रदर्शन और स्केलेबिलिटी प्रदान करता है।

**Q: क्या Aspose.Tasks क्रॉस‑प्लेटफ़ॉर्म विकास को सपोर्ट करता है?**  
**A:** हाँ, आप Aspose.Tasks for Java को किसी भी प्लेटफ़ॉर्म पर उपयोग कर सकते हैं जो Java Runtime Environment (Windows, Linux, macOS) को सपोर्ट करता है।

**Q: क्या मैं खरीदने से पहले Aspose.Tasks आज़मा सकता हूँ?**  
**A:** हाँ, आप एक मुफ्त ट्रायल संस्करण [यहाँ](https://releases.aspose.com/) से डाउनलोड कर सकते हैं।

**Q: Aspose.Tasks के लिए समर्थन कहाँ प्राप्त कर सकता हूँ?**  
**A:** आप समर्थन [Aspose.Tasks फ़ोरम](https://forum.aspose.com/c/tasks/15) पर पा सकते हैं।

## Conclusion
अब आप Aspose.Tasks for Java का उपयोग करके MS Project फ़ाइल से **मुद्रा** अंकों को कैसे प्राप्त करें, जानते हैं। प्रक्रिया सरल है: प्रोजेक्ट लोड करें, `project.get(Prj.CURRENCY_DIGITS)` कॉल करें, और परिणाम को अपने एप्लिकेशन में उपयोग करें। इसी पैटर्न के साथ अन्य प्रोजेक्ट गुणों का अन्वेषण करने में संकोच न करें, और इस लॉजिक को बड़े रिपोर्टिंग या माइग्रेशन समाधान में एकीकृत करें।
