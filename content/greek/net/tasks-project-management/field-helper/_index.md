## Εισαγωγή

Το Aspose.Tasks for .NET είναι ένα ισχυρό API που επιτρέπει στους προγραμματιστές να εργάζονται απρόσκοπτα με τα αρχεία Microsoft Project στις εφαρμογές τους .NET. Είτε δημιουργείτε ένα εργαλείο διαχείρισης έργου, είτε ενσωματώνετε τη λειτουργικότητα του έργου στην εφαρμογή σας είτε απλά χρειάζεται να χειριστείτε τα αρχεία του έργου μέσω προγραμματισμού, το Aspose.Tasks παρέχει τα εργαλεία που χρειάζεστε για να κάνετε τη δουλειά αποτελεσματικά.

Στο έργο σας .NET, θα χρειαστεί να εισαγάγετε τους απαραίτητους χώρους ονομάτων για πρόσβαση στη λειτουργικότητα που παρέχεται από το Aspose.Tasks. Δείτε πώς μπορείτε να το κάνετε:

Τώρα που έχετε ρυθμίσει το Aspose.Tasks στο έργο σας, ας εξερευνήσουμε πώς να χρησιμοποιήσετε το Field Helper για εργασία με πεδία MS Project.

## Βήμα 1: Πρόσβαση σε τίτλους πεδίων εργασιών

 Για να ανακτήσετε τον τίτλο για συγκεκριμένα πεδία εργασιών στο MS Project, μπορείτε να χρησιμοποιήσετε το`FieldHelper.GetDefaultTaskFieldTitle` μέθοδος. Εδώ είναι ένα παράδειγμα:

```csharp
Console.WriteLine("Title for Tsk.ActualCost: " + FieldHelper.GetDefaultTaskFieldTitle(Tsk.ActualCost.KeyType));
```

 Αυτή η γραμμή κώδικα θα εκτυπώσει τον τίτλο για το`ActualCost` πεδίο στην κονσόλα.

## Βήμα 2: Ανάκτηση Ποσοστό Ολοκληρωμένου Τίτλου Εργασίας

 Ομοίως, μπορείτε να ανακτήσετε τον τίτλο για το`PercentWorkComplete` πεδίο χρησιμοποιώντας το`FieldHelper.GetDefaultTaskFieldTitle` μέθοδος:

```csharp
Console.WriteLine("Title for Tsk.PercentWorkComplete: " + FieldHelper.GetDefaultTaskFieldTitle(Tsk.PercentWorkComplete.KeyType));
```
